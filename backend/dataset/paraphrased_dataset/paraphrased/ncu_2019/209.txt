本實驗組合單一樣本選取與單一離散化方法，採用先選取樣本後離散化的順序。首先，將資料集分成80%訓練資料和20%測試資料。接著，使用三種樣本選取方法(DROP3、IB3、GA)篩選訓練資料，減少樣本數量。然後，使用兩種離散化方法(MDLP、ChiMerge)處理篩選後的訓練資料，並記錄切割點，再以相同的切割點離散化測試資料。最後，使用KNN建立模型，以離散化後的訓練資料訓練模型，並用離散化後的測試資料評估模型，透過五折交叉驗證計算平均評估指標。實驗包含六種方法組合：(DROP3 + MDLP + KNN)、(DROP3 + ChiMerge + KNN)、(IB3 + MDLP + KNN)、(IB3 + ChiMerge + KNN)、(GA + MDLP + KNN)和(GA + ChiMerge + KNN)。流程圖如圖3-6和圖3-7所示，虛擬碼如圖3-8所示，其描述了五折交叉驗證中每一折的資料處理流程，包含樣本選取、離散化、模型訓練和評估。
