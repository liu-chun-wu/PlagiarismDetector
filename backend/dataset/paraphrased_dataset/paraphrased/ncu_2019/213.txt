R 語言源於 AT&T 貝爾實驗室於 1970 年代基於 S 語言開發的免費統計軟體，主要應用於資料處理、統計分析、模擬、科學運算與繪圖等領域。 作為開放原始碼軟體，R 語言允許任何人修改原始程式碼或擴充功能。其特色包含向量與陣列導向的矩陣運算、完整的套件與函數功能、強大的繪圖能力、支援 C 語言與 Python 等外部程式以及豐富的教學文件。因此，本研究使用 RStudio 作為開發工具，並利用其內部套件實現資料前處理和機器學習演算法，特別是在資料離散化和分類模型建立階段。


MATLAB 是由 MathWorks 公司於 1984 年推出的教學軟體，名稱源自 Matrix Laboratory（矩陣實驗室）。MATLAB 初期用於矩陣運算，後因應科學視覺化需求，發展成為演算法開發、資料視覺化、資料分析與數值計算的高階計算語言和互動式環境。如今，MATLAB 已成為一套成熟的商用數學軟體。本研究使用 MATLAB 撰寫 DROP3、GA 與 IB3 等樣本選取演算法，主要考量其豐富的數學函式庫。
