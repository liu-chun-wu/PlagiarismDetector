遞迴最小平方估計法(RLSE)由最小平方估計法(LSE)延伸而來。Jang等人指出，線性模型輸出y可表示為：y = θ₁f₁(h⃑) + θ₂f₂(h⃑) + ... + θᵣfᵣ(h⃑) + e，其中h⃑為輸入向量，fᵢ(h⃑)為h⃑的已知函數，θᵢ為待估計參數，e為誤差值。此式可改寫為矩陣形式：Aθ + e = y，其中A、θ、y、e分別為：A = [f₁(h⃑₁), ..., fᵣ(h⃑₁); ...; f₁(h⃑ₘ), ..., fᵣ(h⃑ₘ)]，θ = [θ₁, ..., θᵣ]ᵀ，y = [y₁, ..., yₛ]ᵀ，e = [e₁, ..., eₛ]ᵀ。最小平方估計法旨在找到使誤差平方和最小化的θ。加入下標k表示資料對數量，利用已知θₖ求取θₖ₊₁的方法稱為遞迴最小平方估計法，定義如下：Pₖ₊₁ = Pₖ - Pₖaₖ₊₁aₖ₊₁ᵀPₖ / (1 + aₖ₊₁ᵀPₖaₖ₊₁)，θₖ₊₁ = θₖ + Pₖ₊₁aₖ₊₁(yₖ₊₁ - aₖ₊₁ᵀθₖ)，其中[aₖᵀ; yₖ]代表矩陣[A; y]的第k列，I為單位矩陣。初始設定θ₀為零向量，P₀ = αI，α為極大值。
