Testrpc 是一個基於 Node.js 開發的以太坊客戶端，它將區塊鏈數據暫時儲存在內部存儲空間中。Testrpc 模擬 Geth 客戶端的行為，包含所有 RPC API。發送給 Testrpc 的交易會被立即處理，無需等待區塊挖礦和生成，使基於以太坊的開發測試更加便捷快速。此外，Testrpc 內置 Javascript 版本的以太坊虛擬機 (EVM)，可以完整執行智能合約。通過 npm 指令安裝 Testrpc 後，它會生成十組測試用的區塊和對應的私鑰，並顯示運行端口號。每個帳號預設有 100 個測試用以太幣 (Ether)。需要注意的是，Testrpc 僅運行在內存中，每次重啟都會重置到初始狀態。
