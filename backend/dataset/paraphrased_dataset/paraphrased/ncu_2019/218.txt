本研究探討樣本選取與離散化順序對KNN分類器效能的影響，評估指標包含平均正確率、二元分類資料集的AUC，以及多分類資料集的Kappa。

就平均正確率而言，「先離散化後選取樣本」略優於「先選取樣本後離散化」，但差距不大（不到0.5%）。不過，DROP3搭配MDLP或ChiMerge時，「先選取樣本後離散化」的正確率反而較高，尤其DROP3搭配ChiMerge的差距更超過1.5%。整體而言，「先選取樣本後離散化」的平均正確率略高於「先離散化後選取樣本」約0.45%。表現最佳的組合為DROP3 + ChiMerge + KNN和DROP3 + MDLP + KNN。


針對二元分類資料集的AUC，「先選取樣本後離散化」的平均AUC較高，約高出1.9%。其中，IB3 + MDLP + KNN表現最佳，其次為DROP3 + MDLP + KNN，最後為GA + MDLP + KNN。

對於多分類資料集的Kappa，「先離散化後選取樣本」的平均Kappa較高，約高出1.21%。其中，ChiMerge + DROP3 + KNN表現最佳，其次為MDLP + GA + KNN，最後為MDLP + IB3 + KNN。
