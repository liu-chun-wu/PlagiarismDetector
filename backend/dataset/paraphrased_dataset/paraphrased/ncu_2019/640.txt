特定領域語言 (DSL) 旨在解決特定領域的問題，提供快速有效的應用程式開發方式。許多電腦語言都屬於 DSL，而非通用語言，它們也被稱為應用導向語言、特殊用途語言、專用語言或任務導向語言。DSL 專注於軟體系統的特定面向，不像通用語言能涵蓋所有領域。雖然 DSL 無法構建完整程式，但精心設計的 DSL 比傳統函式庫更易於編寫和修改程式，進而提升程式設計師效率，並改善與領域專家的溝通，有助於解決軟體開發中的難題。常見的 DSL 例子包括用於詞法分析和語法分析的 Lex 和 Yacc、用於文字處理的 Perl、用於硬體描述的 VHDL、用於文件排版的 TeX 和 LaTeX、用於文件標記的 HTML 和 SGML、用於 GUI 的 Tcl/Tk、用於 3D 圖形的 VRML 和 OpenGL，以及用於電腦輔助設計的 AutoLisp 和 AutoCAD。簡而言之，DSL 使用自定義語法處理重複事件，產生預期的程式碼行為。DSL 的優點在於簡化程式編寫、設計和修改，好的 DSL 比傳統高階語言更易於領域專家學習和使用。
