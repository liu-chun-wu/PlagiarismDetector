研究使用根均方誤差（RMSE）作為訓練演算法的指標，RMSE 值越小，預測準確度越高。實驗評估也使用 RMSE，實驗二額外加入平均絕對誤差（MAE）、平均絕對值誤差率（MAPE）和 Theil 不等式係數（Theil U）作為評估指標。MAE 和 MAPE 值越小，預測越準確。Theil U 值介於 0 到 1 之間，越接近 0 表示預測效能越好，反之則越差。由於目標值與輸出值為差分值，為求公平性，以 𝑦𝑖 代表第 𝑖 筆實際目標值，𝑦̂𝑖 代表第 𝑖 筆模型輸出值，𝑛 代表總資料筆數，計算公式如下：

RMSE = √(∑(𝑦𝑖−𝑦̂𝑖)² /𝑛),  i=1 到 n

MAE = ∑|𝑦𝑖−𝑦̂𝑖| /𝑛, i=1 到 n

MAPE = ∑(|𝑦𝑖−𝑦̂𝑖|/|𝑦𝑖|) /𝑛, i=1 到 n

Theil U = √(∑(𝑦𝑖−𝑦̂𝑖)²/𝑛) / (√(∑𝑦𝑖²/𝑛) + √(∑𝑦̂𝑖²/𝑛)), i=1 到 n
