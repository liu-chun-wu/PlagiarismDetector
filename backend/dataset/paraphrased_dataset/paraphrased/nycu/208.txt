選用 DJI Mavic Pro 無人機，主要考量其穩定性。相較於小型無人機，Mavic Pro 在空中懸停等基本技術上更為成熟。起初設計系統時，曾考慮使用自組無人機搭配 MultiWii 開放程式碼，但實作後發現，改良 MultiWii 以實現穩定姿態控制和自動飛行並不容易。加上第一版使用的 ArkLab 飛龍在天無人機抗風性和續航力不足，因此第二版改用具備開放 API 的大廠無人機 DJI Mavic Pro，並將系統設計得更具彈性，使感測器模組能與無人機分開使用。

DJI Mavic Pro 於 2016 年 9 月推出，是一款可摺疊、便攜性高的消費級無人機。它具備 7 公里高清圖傳、視覺與超音波環境感知系統、4K 相機和三軸機械雲台，重量 743 公克，最快速度約 65 km/h，續航時間約 21 分鐘，抗風性良好。即使掛載 310 公克的樹莓派感測模組，飛行姿態依然穩定。

感測器系統採用樹莓派 3 Model B，考量其內建 WiFi、藍牙等常用模組，能提升使用彈性。系統以 Python 程式為主，利用 Linux 的 Crontab 功能，每分鐘上傳感測器資料到雲端資料庫，或收集資料並寫入資料庫，以偵測無人機任務是否開始。感測器模組包含 DHT11 溫濕度整合模組、NEO-6M 導航衛星定位模組、樹莓派本體以及樹莓派鋰電池擴展版。
