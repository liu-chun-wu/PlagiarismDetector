1. **買方尾部力道 (Single Print Buyer Tail Force):**  市場輪廓圖最低價單一價格區間 (Single Print) 的成交量乘以該價格與 TPO 最高價的價差。長尾代表買方強烈反應，推升價格快速上漲。至少兩個單一價格區間組成的尾部才有意義。

2. **賣方尾部力道 (Single Print Selling Tail Force):** 市場輪廓圖最高價單一價格區間的成交量乘以該價格與 TPO 最低價的價差。

3. **買方價格擴張力道 (Buyer Range Extension Force):** 初始平衡區間 (Initial Balance) 以上價格區間的成交量乘以該區間的價格幅度。

4. **賣方價格擴張力道 (Seller Range Extension Force):** 初始平衡區間以下價格區間的成交量乘以該區間的價格幅度。

5. **市場輪動因子 (Rotation Factor, RF):**  用於量化市場力量。若某時段最高價高於前一時段最高價，則賦值 +1；若低於，則賦值 -1。最低價的判斷規則相同。若最高/最低價相同，則賦值 0。將每個時段的賦值加總，得出該交易日的市場情緒。

6. **成交量 (Volume):**  單位時間內的價格成交量，例如每分鐘的成交量，用於表示市場交易的能量。
