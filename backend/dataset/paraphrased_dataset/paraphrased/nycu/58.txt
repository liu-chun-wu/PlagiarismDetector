為了讓使用者在各種裝置上都能方便地使用系統進行機器學習分析，本研究採用網頁為基礎的平台架構。網頁平台的優點包括：免安裝軟體及應用程式、降低裝置環境造成的錯誤、跨平台通用性高，以及良好的使用者體驗。然而，網頁平台也存在一些缺點，例如需要學習多種程式語言，以及多語言整合的複雜性，導致開發過程較為繁瑣。儘管如此，網頁平台易於操作的特性仍受資料分析系統開發者青睞。

為提升系統的模組化和維護效率，本研究將系統分為四個層級：Web Layer、Service Layer、Transform Layer 和 Model Layer。Web Layer 負責視覺化呈現、參數設定及將使用者設定傳遞至 Service Layer。資料交換採用 JSON 格式。Service Layer 接收請求後，將使用者參數存入資料庫，並傳遞資訊至 Transform Layer。Transform Layer 將 JSON 請求轉換為 Python 物件，傳遞至 Model Layer 進行資料處理和模型建立。系統軟體架構和硬體架構分別如圖 3-1 和圖 3-2 所示。
