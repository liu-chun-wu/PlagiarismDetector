長短期記憶（LSTM）旨在解決序列模型中的長期依賴性問題。LSTM將每個時間步的數據處理視為一個記憶單元，並改變了每個單元中信息流動的方式。它通過輸入閘、輸出閘和忘記閘三種閘門，選擇性地控制信息的讀入、輸出和存儲比例，使循環神經網絡兼具長期和短期記憶。LSTM的記憶單元架構可用以下公式表示：

it = σ(Wix_t + Uis_{t-1} + bi)

ft = σ(Wfx_t + Ufs_{t-1} + bf)

ot = σ(Wox_t + Uos_{t-1} + bo)

zt = g(Wzx_t + Uzs_{t-1} + bz)

ct = ft⊙c_{t-1} + it⊙zt

st = ot⊙h(ct)
