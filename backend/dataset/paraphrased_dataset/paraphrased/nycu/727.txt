IEEE 802.11 定義了兩種基本網路架構：Ad-Hoc 模式和 Infrastructure 模式。Ad-Hoc 模式由數個位於同一小區域的 Stations (STAs)，例如筆記型電腦，組成 Independent Basic Service Set (IBSS)，以點對點方式通訊，無需 Access Points (APs)。這種模式組建簡單快速，方便使用者交換資訊。Infrastructure 模式中，同一個 Basic Service Set (BSS) 內的 STAs 共享透過 APs 連接至 Distribution System (DS) 的網路頻寬，STA 數量越多，每台 STA 可用頻寬越小。企業環境通常因範圍廣、使用者多，會建置多個 BSS，並透過 DS 連接形成 Extended Service Set (ESS)。Infrastructure 模式是企業最常使用的模式。
