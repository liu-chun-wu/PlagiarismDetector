設待產天數為7天(Da)，哺乳天數為28天(Dl)，離乳後分娩欄消毒淨空天數為7天(Dc)。分娩欄使用週數(wg)計算如下： wg = (7 + 28 + 7) / 7 = 6週

已知分娩欄總數為350欄，分娩率為80%。每批次最多可容納的分娩母豬數量為218頭 * 80% = 174頭。由於分娩欄使用週數為6週，因此需要準備2批次分娩欄。

假設每名員工可飼養40頭分娩母豬(Hg)，且每批次待配母豬數為214頭，分娩率為80%。則每批次分娩母豬數為214頭 * 80% = 171.2頭。因此，需要 171.2 / 40 ≈ 4.3 名員工。考慮到至少需要1名資深員工(ls)，因此lg + ls >= 5，ls >= 1。
