二值化是將灰階影像轉換為黑白二值影像的分割方法，將大於閥值的灰階值設為白(灰階最大值)，小於閥值的設為黑(灰階最小值)。閥值算法分為固定閥值和自適應閥值兩種。固定閥值由程式或使用者指定，自適應閥值則由程式根據影像計算。本文大量使用固定閥值，並在影像前處理中使用自適應閥值的Otsu算法偵測絲餅面，過濾背景。

Otsu算法(最大類間差法、大津演算法)被認為是影像分割中最佳的閥值選取算法。其原理是使前景和背景的類間方差最大化，減少錯分。對於大小為𝑀×𝑁的影像，閥值為T，小於T的像素數量為𝑁0，大於T的像素數量為𝑁1。前景像素比例為𝜔0，平均灰度為𝜇0；背景像素比例為𝜔1，平均灰度為𝜇1；影像總平均灰度為𝜇，類間方差為𝑔。透過掃描T(0~255)，找到使𝑔最大的T值。類間方差𝑔的計算公式如下：

𝜔0 = 𝑁0 / (𝑀×𝑁)
𝜔1 = 𝑁1 / (𝑀×𝑁)
𝑁0 + 𝑁1 = 𝑀×𝑁
𝜔0 + 𝜔1 = 1
𝜇 = 𝜔0 * 𝜇0 + 𝜔1 * 𝜇1
𝑔 = 𝜔0 * (𝜇0 - 𝜇)^2 + 𝜔1 * (𝜇1 - 𝜇)^2
𝑔 = 𝜔0 * 𝜔1 * (𝜇0 - 𝜇1)^2

使用Otsu算法獲得閥值後，即可生成二值影像，有效分割絲餅面和背景，避免程式計算背景，提升速度並減少誤判。

本章介紹兩種檢測外溢瑕疵的方法：基於光影變化的Shadow Scan Method和基於梯度的Dominant Gradient Analysis。兩種方法都將絲餅影像切割成固定大小區塊，分析區塊特性，並使用第二章的影像前處理輔助檢測。
