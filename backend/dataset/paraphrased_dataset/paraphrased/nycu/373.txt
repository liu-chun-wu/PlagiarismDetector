實驗結果將使用多種圖像品質測量指標評價，包括傳統的全參考指標如峰值信噪比 (PSNR) 和均方誤差 (MSE)，以及考慮人類視覺系統 (HVS) 的指標，如結構相似度指數 (SSIM)、特徵相似度 (FSIM)、梯度幅度相似性偏差 (GMSD)、視覺信號保真度 (VIF) 和信息保真度準則 (IFC)，以測量圖像相似程度。


美化QR code 的產生步驟如下：首先，利用zxing 應用程式介面(API)產生包含網址資訊的QR code，並參考文獻[14]放大QR code，將每個點處理成5x5像素。接著，將圖片進行小波轉換，並參考文獻[20]使用Sobel運算子提取圖片邊緣，與預先建立的去主題遮罩比較，找出相似度最高的主題遮罩。最後，將目標圖像嵌入經過前兩個步驟處理的QR code 中，使QR code兼具解碼和圖像顯示功能，更美觀且貼近生活。
