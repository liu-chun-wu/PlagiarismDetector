使用者想在某地找到餐廳，便打開手機App搜尋比較後決定用餐地點。

使用Google Map的步驟：
1. 打開 Google Map。
2. GPS 定位目前位置。
3. 搜尋附近的餐廳。
4. 查看搜尋結果，並依距離由近到遠點擊查看店家評價，直到找到滿意的餐廳。

使用本系統的步驟：
1. 打開本系統。
2. 設定搜尋條件（餐廳，500公尺內），並儲存。
3. 點擊搜尋。
4. 查看搜尋結果，並依照靜態地圖上標示的推薦順序查看。

本系統與Google Map的比較：

| 比較項目 | Google Map | 本系統 |
|---|---|---|
| 搜尋類型 | 多種類型 | 目前僅提供旅遊相關類型 |
| 搜尋結果畫面表現方式 | 平面地圖 | 平面地圖+擴增實境 |
| 搜尋速度 | 快 | 慢（因系統額外進行篩選、運算及排序） |
| 尋找喜愛地點所需時間 | 搜尋結果多時，需花費較多時間比較 | 快速，依系統推薦順序查看即可 |
| 搜尋結果呈現 | 無法直覺得知搜尋結果與目前位置的方向及距離 | 可以直覺得知搜尋結果與目前位置的方向及距離 |
| 景點推薦 | 無直接顯示景點評分 | 利用Google評分排序推薦景點 |
| 其他功能 | 關鍵字搜尋、導航、評價資訊、景點相片等 | 顯示目前方向角度 |
