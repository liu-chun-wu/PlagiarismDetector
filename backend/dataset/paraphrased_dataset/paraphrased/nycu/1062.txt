「附近地點搜尋」功能可依關鍵字或指定類型搜尋指定區域內的相關地點，縮小搜尋範圍。API 介面為：https://maps.googleapis.com/maps/api/place/nearbysearch/output?parameters，其中 output 可設為 json 或 xml，指定回傳格式。parameters 為傳入參數，必要參數如下：1. key：應用程式的 API 金鑰；2. location：指定中心點的緯度/經度，格式為「緯度,經度」；3. radius：設定搜尋半徑（公尺），最大值為 50,000 公尺。搜尋結果將依據 URL 中 output 的設定格式返回。JSON 格式的根元素包含：1. status：與請求相關的中繼資料；2. results：包含地點陣列及其資訊；3. html_attributions：需顯示給使用者的資料引用標示；4. next_page_token：用於獲取最多 20 個額外結果的語彙基元，最多可返回 60 個結果，若無額外結果則不返回此欄位。（圖 10 為「附近地點搜尋」服務的部分回應值範例，資料來源：Google Places API）
