為研究繪圖手眼協同控制技巧，受試者在數位（RWI、VRI、VREI）和傳統紙筆條件下描繪五種圖片輪廓（魚、人物、烏龜、雨傘、房子），這些圖片輪廓長度、曲度和交叉複雜度各異。研究比較了不同實驗條件下的描繪結果。由於數位和紙筆條件的解析度和線寬不同，採用類似於書寫任務的影像骨架化方法，將線寬標準化，確保結果可比較。非輪廓像素被設為誤差像素，計算其到輪廓的垂直距離（誤差距離），累積所有誤差距離再除以誤差像素數量，得到平均誤差距離。ANOVA模型設計為（實驗條件）X（圖片類型）X（輸入方式）。

結果顯示，圖片類型[F(4, 160) = 29.33, p < 0.001, η² = 0.42]、實驗條件[F(2, 40) = 6.93, p = 0.003, η² = 0.26]和輸入方式[F(1, 160) = 27.68, p < 0.001, η² = 0.41]對平均誤差距離有顯著影響。輸入方式在不同實驗條件下也存在顯著差異[F(2, 160) = 18.79, p < 0.001, η² = 0.48]。圖片類型和輸入方式的交互作用[F(4, 160) = 4.067, p = 0.004, η² = 0.092]，以及圖片類型、輸入方式和實驗條件的三重交互作用[F(8, 160) = 2.05, p = 0.04, η² = 0.093]也達顯著水平。研究重點關注不同實驗條件下輸入方式的影響。

Bonferroni事後分析顯示，RWI條件下數位和紙筆輸入的平均誤差距離無顯著差異。VRI條件下，數位輸入(M=3.56, SD=0.70)顯著大於紙筆輸入(M=3.13, SD=0.28), p = 0.008。VREI條件下，數位輸入(M=3.8, SD=0.56)顯著大於紙筆輸入(M=3.12, SD=0.28), p < 0.001。

Tukey’s HSD事後分析顯示，紙筆輸入在RWI、VRI和VREI條件下無顯著差異。數位輸入方面，VRI(M=3.56, SD=0.65)顯著大於RWI(M=2.93, SD=0.29), p = 0.005；VREI(M=3.82, SD=0.47)也顯著大於RWI, p = 0.001 (見圖17)。圖17左圖為數位輸入，右圖為紙筆輸入的平均誤差距離。
