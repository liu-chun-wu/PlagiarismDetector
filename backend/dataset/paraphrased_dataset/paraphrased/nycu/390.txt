Warcraft Log 的戰鬥紀錄包含戰鬥過程圖、回放功能及各種資訊。本研究將利用爬蟲擷取頭目名稱和難度、戰鬥時間、戰鬥結果（成功或失敗）、團隊人數、成員職業和裝備等級等資訊建立預測模型。

如同體育賽事研究利用得分、籃板等數據建立預測模型，副本研究中，玩家的輸出傷害、承受傷害和治療量是評估表現的關鍵指標。Warcraft Log 也提供 PR 值來比較玩家表現。

然而，本研究不使用傷害量、承受傷害、治療量和 PR 值。因為魔獸世界玩家數量龐大且團隊成員流動性高，難以像 NBA 或 NFL 等職業運動那樣追蹤每個玩家的數據。

因此，本研究將專注於利用裝備和團隊組成建立預測模型。裝備代表坦克的防禦力、補師的治療量和輸出的傷害量，藉此預測戰鬥結果。雖然技術也會影響玩家表現，但由於大多數裝備是拾取綁定，裝備等級某種程度上反映玩家的副本經驗和潛在表現。
