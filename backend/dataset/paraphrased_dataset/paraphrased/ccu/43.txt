Wasserman和Faust (1994) 將社會網絡分析 (Social Network Analysis) 定義為透過觀察社會活動者之間的互動和行為活動，以探索潛在網絡結構的社會學方法。社會網絡分析運用圖論和數學定量方法描述個體間關係，已廣泛應用於電腦科學、數學、心理學、社會學和管理學等領域。  Hanneman (2005) 指出社會網絡由行為者 (Actor)、連結 (Linkages) 和關係 (Relationship) 三要素組成。Goodreau (2007) 則強調資訊科技在社會網絡分析領域的重要性，尤其在處理和觀察大量資料及其相互關係方面。

Scott (2000) 提出的社會網絡分析方法包含：

1. 平均最短路徑 (Average shortest path length)：社會網絡中任意兩點間平均最短距離，用以衡量節點間資訊傳遞效率。

2. 直徑 (Diameter)：社會網絡中任意兩點間最長路徑長度，用以衡量網絡大小。

3. 網絡密度 (Network density)：衡量行為者或節點間的緊密程度，計算公式為：密度 =  R / (n(n-1)/2)，其中 R 為網絡圖中連結數量，n 為網絡圖中節點個數。網絡密度越高，表示相同節點個數下連結數量越高，節點間關係越緊密。

4. 網絡中心性 (Network Centrality)：Cho et al. (2007) 定義為行為者連結在網絡中相對重要性的位置，衡量行為者獲取資源能力和網絡影響力。中心性可分為程度中心性、接近中心性和中介中心性 (Faust, 1997)。

    (1) 程度中心性 (Degree Centrality)：以行為者與其他相鄰節點的數量衡量其在網絡區域的中心程度。公式為：DCi = (∑ aij, j=1 to n) / (n-1) ，其中 n 為網絡中所有節點個數，∑ aij, j=1 to n 為與節點 i 存在連結關係之節點總和。程度中心性越高，表示行為者在網絡中越重要。

    (2) 接近中心性 (Closeness Centrality)：衡量行為者與其他節點間的緊密程度，即訊息接收程度。公式為：CC(ni) = (n-1) / (∑ dij, j=1 to n)，其中 n 為網絡中所有節點個數，∑ dij, j=1 to n  為與節點 i 存在連結關係之節點最短距離總和。接近中心性越低，表示行為者接收節點訊息程度越低。

    (3) 中介中心性 (Betweenness Centrality)：衡量行為者在兩節點或社群間路徑的重要性。公式為：BC(ni) = ∑ (σst(ni) / σst), s≠ni≠t∈V，其中 n 為網絡中所有節點個數，V 為所有節點之集合，σst 為節點 s 至節點 t 最短路徑個數，σst(ni) 為節點 s 到節點 t 最短路徑過程中，經過 ni 的節點路徑個數。接近中心性越高，表示行為者在兩節點或社群間取得較多資源。
