若封包目標存在但未知，則以隨機方式發送，其送達機率稱為訊息轉發吻合機率。訊息轉發吻合機率參數定義如下：

節點轉發封包數量上限：a
封包被轉發層數上限：l
Metadata散佈數量：m
封包被轉發的總數量r，公式如下：
r = 1 + a + a² + ... + aˡ =  (aˡ⁺¹ - 1) / (a - 1) , if a > 1;  l + 1, if a = 1
區域內節點的數量：n
訊息轉發吻合的數量：k
區域內如實回報訊息轉發吻合的節點比例：X
區域內非自私節點的比例：F

訊息轉發吻合機率採用超幾何分配公式計算。若 mx + r ≤ n 與 k ≤ min(mx, r) 不成立，則訊息轉發吻合機率為 0。
