在資料前處理完後，本研究使用Pure Neural Network Model 預測量測值，因為多維 度預測單一維度資料（2300 維製程資料預測1 維量測資料），在輸出層必需為1 個 Neural，並且因預測值為確切數值（非分類型的預測），損失函數（Loss Function）使用 Accuracy（需與目標數值完全一致）無法正確評估模型的Loss，例如：目標為10 而模 型預測值為10.1，雖然與目標僅相差0.1 的距離，但Accuracy 仍然為0，因此針對這種 使用狀況，本研究選擇MAE（Mean Absolute Error）作為Loss Function，以與目標的平 均絕對誤差當作Loss 使模型預測趨近於量測值。 圖3-3 為Pure Neural Network Model 研究流程，資料取得後經由資料前處理後訓練 Pure Neural Network Model，因資料前處理時使資料與預測結果皆為標準化後的結果， 但本研究目的為模擬實時量測結果，因此必需將預測結果反標準化至真實量測數值才為 研究所需的量測結果。 圖3-3 Pure Neural Network Model 研究流程(圖片來源：本研究整理)
