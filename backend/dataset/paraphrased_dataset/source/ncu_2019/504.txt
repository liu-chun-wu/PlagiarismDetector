Remix 是一個基於瀏覽器的Solidity 編譯環境，開發者可以直接透過網路連 線至其線上網站(https://remix.ethereum.org/)進行開發，Remix 為智能合約開發提 供了一個整合的開發環境，透過瀏覽器便可以直接編譯、除錯、部屬、測試與開 發智能合約、此外，它也能夠與web3 、Metamask、testrpc 等相關工具進行調配 與整合。 圖32 Remix 開發環境 資料來源：本研究整理 如圖33 所示，在Compile mode 中，Remix 可以及時編譯Solidity 智能合約， 方便及時發現任何語法錯誤。此外，由於Solidity 各版本有不同支援的功能，因 此，Remix 提供各版本的編譯器供開發者選擇，預設為最新版。 如圖33 所示，在Run mode 中，Remix 提供了JavaScript VM、Injected Web3 與Web3 Provider 三種測試環境，使用者可依自己的需求選擇特定的選項。 JavaScript VM 為Remix 內建的JavaScript 虛擬環境，代表所有資料都是存在本地 記憶體（功能類似 testrpc）。它會提供 5 個虛擬帳戶，每個有 100 ETH，可直接 進行合約部屬。若是選擇Injected Web3，Remix 會自動連結 MetaMask。若選擇 Web3 Provider ，則可以指定要連結的節點位址，例如：本機的私網路 http://localhost:8545。連接上區塊後，可透過下方的使用者介面直接部屬新的合約 或輸入合約地址存取已部屬合約，並呼叫合約方法。 圖33 Remix 開發環境(Compile mode & Run mode) 資料來源：本研究整理
