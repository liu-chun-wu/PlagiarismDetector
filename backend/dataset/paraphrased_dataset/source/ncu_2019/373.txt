球型複數模糊集是將球坐標系（Spherical coordinate system）及複數的概念結合。對 於球坐標系，三維空間中的任意一點位置，可由三個參數來表示，包含徑向距離r、天 頂角θ、方位角φ。透過直角坐標系的轉換，可得到該點在三軸上映射之位置。圖 2 為 球座標系之示意圖。 圖 2：球座標系之示意圖 卷積模糊神經系統於財金市場數據之研究 與球坐標系相似的概念被應用於球型複數模糊集理論中，其歸屬程度的計算方法可 分為三個階段，第一階段為計算振幅值𝑟及多個相位值𝜃，在本研究中，分別使用高斯型 態的歸屬函數（Gaussian type membership function, GMF）以及其偏微分方程式來計算之， 如下式： 𝑟= GMF(ℎ; 𝑐, 𝜎) = exp [−1 2 (ℎ−𝑐 𝜎 )2], (3) 𝜃1 = GMF′(ℎ;𝑐,𝜎) = exp [−1 2 (ℎ−𝑐 𝜎 ) ] ∙[−(ℎ−𝑐) 𝜎2 ], (4) 𝜃2 = GMF′′(ℎ;𝑐,𝜎) = exp [−1 2 (ℎ−𝑐 𝜎 ) ] ∙[(−(ℎ−𝑐) 𝜎2 ) −1 𝜎2], (5) 𝜃3 = GMF′′′(ℎ; 𝑐, 𝜎) = exp [−1 2 (ℎ−𝑐 𝜎 )2] ∙[(−(ℎ−𝑐) 𝜎2 ) −3(ℎ−𝑐) 𝜎2 ], (6) 其中，ℎ為輸入值；𝑐為高斯型態之歸屬函數的中心值；𝜎為高斯型態之歸屬函數的寬度。 第二階段為計算歸屬程度，透過直角坐標系的轉換方式，計算出不同的歸屬程度𝑢，如 下式： [ 𝑢1 𝑢2 𝑢3 𝑢4 ] = [ 𝑟∙sin𝜃1 ∙cos𝜃2 𝑟∙sin𝜃1 ∙sin𝜃2 𝑟∙sin𝜃2 ∙cos𝜃3 𝑟∙sin𝜃2 ∙sin𝜃3 ]. (7) 第三階段為向量化，將不同歸屬程度結合複數的概念，產生出球型複數模糊集之歸屬程 度𝜇⃑，如下式，其中𝑗= √−1： 𝜇⃑= [𝜇1 𝜇2] = [𝑢1 + 𝑗𝑢2 𝑢3 + 𝑗𝑢4]. (8) 球型複數模糊集可根據目標數，在不增加參數數量的情況下，一次產生多個複數歸屬程 度，比起複數模糊集與Type-1 模糊集，能傳遞更多的歸屬資訊。 卷積模糊神經系統於財金市場數據之研究
