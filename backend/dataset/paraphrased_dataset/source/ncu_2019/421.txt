在將Pure Neural Network Model 以各自權重結合Rule Based Improved Neural Network Model 後，因權重為人為決定，無法由各自資料特性決定權重，如此則失去本 研究的目的—以深度學習模型取代人工建置數學模型的初衷，因此在Pure Neural Network Model 和Rule Based Improved Neural Network Model 反標準化後將各自結果與 量測資料訓練Weight Neural Network Model，此權重模型可解決以下二項問題： 1. 人工調整權重 不需再人工調整權重，由模型自行決定 2. 預測結果誤差在反標準化後被放大 在Pure Neural Network Model 和Rule Based Improved Neural Network Model 研 究架構中，因訓練時為標準化後的結果，在訓練時的MAE 會在反標準化後被放大， 導致模型準確度能提升的空間有限，但在Weight Neural Network Model 中，因訓練 資料為反標準化後結果，在預測後不需執行反標準化，因此訓練時的Loss 不會在 結果被放大，可使模型的準確度提升。 加入權重模型後的結果如圖3-6 所示，資料經資料前處理後訓練Pure Neural Network Model 並將人工選出的重要參數送入訓練Rule Based Improved Neural Network Model 在各自經由反標準化後訓練Weight Neural Network Model 再獲得預測結果。 圖3-6 Pure NN 結合Rule Based Improved NN 以Weight NN 結合之研究架構 (圖片來源：本研究整理)
