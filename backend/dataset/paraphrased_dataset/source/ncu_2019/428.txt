經過四個實驗在三種資料處理的狀況下比較五種遺失值處理方法相互比較後，可以 得到以下結論： 1. Weight NN 架構有效使平均誤差降低。 2. 手動權重模型供法有效降低誤差。 3. Rule Based NN 平均誤差較Pure NN 低。 4. 同時去除low variance/極端值平均誤差較低。 5. 去除極端值較去除low variance 對平均誤差影響較大。 6. 以Median 補值(去極端值/low variance)平均誤差較低。 在得到以上實驗結論後可以結果推論，Weight NN 可有效降低預測誤差的原因，推 論為模型加入了Pure NN 與Rule Based Improved NN 的預測值作為經驗再與量測值找出 了更穩定的關係特徵，因此Weight NN 的設計經由實驗證明對降低預測誤差是有幫助 的。而手動權重模型無法有效降低預測誤差，推測原因為Rule Based Improved NN 預測 誤差較小，而Pure NN 預測誤差較大，當手動權重的結果，只會使預測值介於最小與最 大之間，可使預測的誤差範圍縮小，但無法使平均誤差降低，在Rule Based Improved NN 的預測誤差較Pure NN 的預測誤差小的部份，可證明在此實驗的量測值上，專業知識挑 選的參數的確對目標量測參數有一定的關聯性，而去除low variance/極端值可幫助 Neural Network 得到較接近真實數值的預測，而其中的極端值影響大於low variance 影 響，推測此極端值可能為機台誤報的數值，因此在不去除極端值的時候預測結果在各個 實驗中皆有較大差距，而在去除low variance/極端值後，以Median 補值在本研究當中 得到低的預測誤差，由此推論，因已去除low variance/極端值，部份的空值因機台理論 上應常時間維持一樣的數值才能保持穩定的產品製程，所以在此點的實驗結果符合常理 上的邏輯。 表4-17 為各實驗下平均誤差最小的組合，可觀察到與現有VM 數學模型相比，雖 然每一個Neural Network 模型平均誤差皆較低，但輔以前幾個實驗的所有數據比較而 言，手動權重在所有數據上無法顯著使平均誤差降低，但在平均誤差最小的組合部份低 於Pure NN 和Rule Based NN，但並不低於Weight NN，且使用手動權重即需要人工選 擇才能完成此一架構，因此在所有實驗後，以平均誤差和做法比較下，Weight NN 為最 佳的架構。 表4-17 各實驗下平均誤差最小之數據 第五章  結論
