本研究利用程式交易衡量深度學習產生之投資績效，另外衡量深度學習模型訓練 時間。 將資料輸入至MultiCharts，衡量投資績效如淨利、投資年化報酬率、投資年化報 酬率標準差（standard deviation）、最大投資損失（Max portfolio drawdown）、風險報酬 比(Risk reward return rate)及獲利高原。以下說明各項衡量指標。 A. 淨利：越多越好。 B. 年化報酬率：年化後的報酬率，越多越好。 C. 年化報酬率標準差：年化報酬率的標準差，越低越好。 D. 最大投資損失（Max portfolio drawdown）：持有股票，直到下一個高峰來臨前，從 股價的高峰到谷底的損失的最大值(由於風險報酬比已將最大投資損失影響計入， 故最大投資損失、最大投資損失(%)只在附錄1 列示，不於正文列示)。 E. 最大投資損失(%)（Max portfolio drawdown）(%)：最大投資損失/初始投資金額， 為一比率。 F. 風險報酬比(Risk reward return rate)：淨利／最大投資損失，代表每承擔一元最大投 資損失可以獲得多少淨利，為經過風險修正後的報酬，風險報酬比越高，代表承擔 相同風險時，獲利越高，單以年化報酬率比較的話，就算年化報酬率高該年度可能 也有大額的損失，以風險報酬比衡量避免了這個問題。 G. 獲利高原：為了檢驗神經網路是否可產生穩健之投資績效，將神經元數32、128、 512 三種參數組合產生的年化報酬率做平均，得到神經元平均報酬率，若連續有超 過一天持有天數年化報酬率高於買進持有策略的年化報酬率，視為該參數組合可形 成獲利高原。 肆. 研究結果分析 本章節說明本研究設計之股票投資深度學習回測模型實際應用之投資績效，4.1 節 說明環境設定，4.2 節說明本研究設計之系統流程，4.3 節說明本研究設計之系統驗證 結果。
