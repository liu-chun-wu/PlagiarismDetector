基於研究動機所提及之問題，本研究提出以下解決辦法。 問題1：原系統架構僅適用於台灣股市 解決辦法：依據不同股市規則進行結構修改 考慮各股票市場之差異性，例如台灣財報時間點為季、美國有季或年，修改系統 結構以套用在不同股票市場上。 問題2：投資組合模型變數難以擴充 解決辦法：建立可擴充式投資組合績效回測結構 運用物件導向概念，將原本系統進行全面重構，整合原先已有的基本面等分法切 割、技術面交易策略、資金配置法、最大持有股數等多種變數，並新增「資金再投入 方法」為新的變數，建構一個可擴充式投組績效回測結構，如下圖 1 所示，利用可高 度客製化的程式交易平台，針對不同股票市場多種投組、多種商品進行自動化回測。 此可擴充式模型能夠經由客製化財報時間點處理，再延伸至其他股票市場使用。 圖 1、可擴充式投組績效回測結構 問題3：仍需手動開啟Amibroker 執行batch 檔進行回測 解決辦法：改善自動計算投組績效流程 利用 Python 程式設計與 Amibroker 可客製化之回測功能，建立多商品、多策略 自動化績效回測流程，以Python 直接呼叫Amibroker 進行回測，取代手動程序，提高 系統效率。 問題4：現有回測結果呈現為靜態且不直覺 解決辦法：整合數據並分析，使用新的平台呈現視覺化回測結果 整合所有變數進行量化分析，透過 Python 分析資料並以視覺化圖表於Jupyter Notebook 呈現分析結果，解決過往投資人難以理解交易回測結果報表的問題，能更輕 鬆評估交易投組與策略。
