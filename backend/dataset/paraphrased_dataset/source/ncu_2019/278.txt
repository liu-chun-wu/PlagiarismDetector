(1)  教育訓練需求與目標 對於行員的程式語言教育訓練，提升行員能力是本方案重要的目標，而對此 提出5 點的需求，第一點是發揮有教無類、教育公平的「公平的受教機會」，第 二點是能發揮因材施教的「多元學習方式」，第三點是強調實務可行性的「有限 資源內讓學習效果最佳」，第四點是發揮學以致用、 PBL 的「能在銀行業務上運 用」，第五點是強調教學相長、 PBL 、啟發教學的「有交流管道來教學相長」， 以下並就此5 點在本方案進行分析： 1. 公平的受教機會： 首先是公平的受教機會，讓每位行員的受教機會是均等，皆有學習的機會， 尤其是初階入門的學習，要不論每位行員的先備知識、能力、學歷、經歷等 皆需有相同的機會，而若是在規劃進階課程有名額限制時，錄取名額的條件 與資格也要是所有行員都有機會去努力爭取的，如此才能維持公平的組織文 化。 2. 多元學習方式： 需要有多元的學習方式，因為對象是所有約6,000 名行員，每個行員的能力 高低程度、先備知識、能學習的時間、適合的學習方式等皆可能有差異，所 以建立越多元的學習方式，能讓行員學習的方式與管道越多，才越可以配合 每個行員不同的適合方式。 3. 有限資源內讓學習效果最佳： 良好的規劃需能在實務上以有限資源內完成，否則也僅是紙上談兵，如對 6,000 名行員進行100 小時目前時下熱門的 C++ 或 Python 程式語言教育 訓練，時數長讓學習成效提升，熱門的程式語言更符合時代要求，但在實務 上的資源是遠遠不足來實行的，只能輪為空談；所以能在有限資源內實行是 非常重要的，依 L 銀行現況來看，需安排資源能負擔的教育訓練時數，並 需設計在短時間內易學的學習內容，以及易用的設計才能讓行員透過問題解 決來學習與累積經驗，而將能運用的資源皆做最有效用的發揮運用，才能讓 學習效果達到最佳。 4. 能在銀行業務上運用： 教育訓練除了提升行員的能力外，最重要的即是要讓行員能學以致用，在業 務上能有所發揮與運用，才能達到最好的學習效果與學習的意義。本方案是 以 PBL 來進行程式語言教育訓練，讓行員在思考問題與解決問題中學習， 而教材與課程設計皆以銀行相關業務為內容，不只是因為能讓行員比較熟悉 內容而提升學習效果，並能讓行員學習如何將學習的程式語言運用在業務 上，在教學程式語言的同時也在對實際運用進行教學，來讓行員能達到學以 致用的目標。 5. 有交流管道來教學相長： 在學習時交流與討論是非常重要的，尤其是在 PBL 裡對於如何解決問題的 討論，而 L 銀行的行員是散佈在全臺各地，使當面交流不易，但若能建立 良好的網路交流管道，則所有行員不再受到地點的限制，讓總行與行員間， 以及行員與行員間，皆能互相交流、教學等，提問的行員能獲得解答，回答 與參與交流討論的行員，能經由探討問題與衍生思考等來有所收獲，達到良 好的教學相長。 以上整理成如表4-3 所示。 表4-3  教育訓練需求與目標 項目 內容 １. 公平的受教機會 讓每位行員的受教機會均等，皆有學習的機會，尤其是 初階入門的學習。 ２. 多元學習方式 提供多元的學習方式供行員去選擇與使用，以配合每 個行員不同的學習適合方式。 ３. 有限資源內讓學習 效果最佳 銀行可運用的方式，如練習用的 教材、行訓所的教育訓練、線上課程 E-Learning 、分行視訊課程、教學文章、 論壇交流等，首先是教材上，不只能在上課時增進學習的效果，在課後練習與經 驗累積等也有很大的幫助，而程式語言學習是非常注重經驗的，所以建立良好的 教材對於學習是至關重要的；在行訓所的現場教育訓練，是良好的方式，也是本 行對於重要業務訓練的做法，不過受制於經費預算等相對高昂，所以時數與次數 皆受到限制，需在有限的時數內達到良好的教學效果；線上課程 E-Learning 也 是良好的教學工具，可以讓行員隨時進行學習，並可視程度與學習效果選擇適合 的課程，也可成為行訊所現場教育訓練後的複習課程，以及讓有興趣的行員可以 擴充深度與廣度，所以需建立完整且適合各種程度的線上課程；在 L 銀行除了 行訊所、行員電腦可以進行學習外，各分行也有視訊設備可供教育訓練使用，所 以將分行視訊課程也列為程式語言的教育訓練工具之一，在有需要時編排課程， 也是可以增加學習效果的，使教育訓練的管道多一種方式；教學文章也是個良好 的學習方式，因為每個人適合的學習方式不盡然相同，所以建立多元的學習方式 供行員去選擇與使用，才能最大的提升學習效果；而交流在程式語言學習上也是 非常重要的，所以建立論壇讓行員可以進行交流與學習，讓行員在編寫程式遇到 問題時能得到協助，或是從別人的問題中得到啟發等，對於程式學習皆是非常有 幫助的。 在各種教育訓練方式中，在行訓所的現場教育訓練是很重要的一項，因為現 場講授課程的學習效果是較佳的，本方案以3 小時為初階的程式語言教育訓練時 數規劃，依現有的 L 銀行教育訓練環境與制度，上課時數主要分為半日的3 小 時與整日的7 小時兩種，而在人力資源等條件限制下，半日3 小時的課程仍能讓 行員有另外半日處理工作業務，既能讓行員受訓學習並將業務影響降至最低，對 全體約6,000 名員工進行3 小時教育訓練是尚在可實行的範圍。以每堂課供80 名學員，每週一堂共需75 週，約一年半可對全體員工訓練完畢，另新進員工有 2 星期之行訓所課程受訓，安排從中插入3 小時程式語言課程，如此便可讓所有 行員皆接受過初階的3 小時教育訓練。並規劃中階、高階等教育訓練，並視初階 課程實行與學習成效去規劃課程內容，本研究僅就初階教育訓練進行討論，待成 效佳時再進行中高階教育訓練之探討。 (3)  3 小時教育訓練教學內容 針對初階3 小時教育訓練的上課內容設計，本方案是以讓初學者的行員能快 速上手，並能進行問題解決與處理的程式編寫，以及能運用教材進行練習為主。 分為基本教學、運用方式與規範宣導、教材實例教學與運用方式等三個部分，第 一部分的基本教學先以程式概念開頭，講述基本的程式概念，接著教學錄制巨集， 錄製巨集是讓初學者能快速學習程式語言的方式，能將行員在 Excel 執行的動 作以程式語言來呈現，行員可以以此來進行學習與修改程式，讓學習速度快速邁 進；然後是 VBA 與 Excel 配合運用，因為 VBA 在編寫程式碼時，不一定所有 運算皆需以 VBA 來編寫，也可以運用行員熟悉的 Excel 的公式或函數等來運 算，如此可讓行員不需累積太多的 VBA 程式語言的學習，即可編寫許多的程式 來執行；下一項是教學基本控制流程迴圈、判斷、變數，因以運用為主，故在數 種的迴圈與判斷中僅各擇一樣來教學，而因行員的先備知識是熟悉 Excel ，判斷 與變數在 Excel 中有類似函數與運用，行員能較快理解，所以以迴圈為主要教學 項目；最後再對教材與實務上的常用例句進行介紹，以及說明幾種程式執行的方 式。 第二部分是運用方式與規範宣導，向行員介紹本方案的規劃與運用方式，如 教材下載與運用、論壇運用方式、提案方式與管道、獎懲制度與其他如 E-Learning 等的學習方式，讓行員了解整個方案並能有效運用，並針對 EUC 管理進行規範 宣導，讓行員了解資安的重要與 EUC 需遵守的要項。 第三部分是主要的教材實例教學與運用，本方案的教學方式以 PBL 為主， 所以透過針對行員熟悉並已在使用的 Excel 教材進行程式編寫與問題解決方式 教學，以此來學習教材的程式編寫方式及如何運用程式語言來解決問題，讓行員 能學習編寫程式語言，也讓行員能了解教材的程式編寫並能創造出更多業務上的 運用，以及能去思考更多的問題與解決方式。
