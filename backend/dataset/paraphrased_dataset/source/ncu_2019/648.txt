應用LabVIEW 進行設計負載潮流分析，LabVIEW 提供控制元件、指示元件與常數 元件，測量、儀器、信號、數據通訊……等多樣化的工程元件，以供DSL programmer 用 來建構模擬工程系統，LabVIEW 實作負載潮流時大致會需要用到控制元件，資料型態 轉換的元件，電力工程元件，藉由這些組合出分析的演算法，引入數據和運行計算的界 面，模擬系統的控制結果。 對於使用LabVIEW 與其元件進行開發的架構，本研究亦依開發者與使用者的對象 不同，以如下層面進行分析與討論。如圖2.3.1.a，以設計開發power flow 分析的DSL 為 例，LabVIEW 開發一套DSL 的流程，會是DSL developer 藉由LabVIEW 這套DSL framework 建構一套應用於power system 分析工具的DSL，讓DSL programmer 可以藉 由在LabVIEW 的Block Diagram 介面上使用該套DSL 拖拉建構program，如圖2.3.1.b， 最後提供給End-User 在Front Panel 上操作這個program。 圖2.3.1.a  LabVIEW 操作與開發架構圖 圖2.3.1.b  Graphical-program-of-DC-load-flow-VI-using-LabVIEW[17]
