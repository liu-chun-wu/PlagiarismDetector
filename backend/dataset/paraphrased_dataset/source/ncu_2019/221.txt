近幾年電腦運算能力不斷地增強，資訊技術的快速發展也帶動了資料探勘技 術的蓬勃，人們透過資料探勘來幫助決策、獲取有價值的知識。但資訊科技的發 展也讓資料集的樣本數量日益龐大，此現象可能會造成雜訊充斥，影響最後模型 的準確度與效能；此外，資料集中常會包含不易判讀的連續型數值，這類型的數 值本身複雜的特性，亦為影響最後模型準確度的因素之一。從過去的經驗中可以 得知資料前處理的重要性，透過適當的資料前處理，不僅可以減少雜訊以及增加 資料判讀性，更可以提升最後模型的預測準確度，確保我們在進行資料探勘時所 使用的資料品質。 樣本選取與資料離散化，皆已經被證實能夠有效解決雜訊以及增加資料判讀 性，讓最後的模型準確度能夠有所提升，本研究將此兩種前處理方法做結合搭配， 期望透過兩種資料前處理方法的綜合運用，來讓最後的模型指標能夠再有所提升。 本論文提出了IS + D + KNN 與D + IS + KNN 兩種架構，分別從中選用了三種樣 本選取方法（IB3、DROP3 與GA）與兩種監督式離散化方法（MDLP 與ChiMerge）， 並從中探討何種結合方式與其先後順序，能夠得到比已離散過後所得到的模型指 標，還要更為提升的數值，並除了探討平均正確率外，也針對二分類與多分類資 料集常用的衡量指標：AUC 與Kappa，做詳細的比較與討論，找出這些架構最適 合運用的情境。在本實驗結果驗證過後，提出以下五點相關的研究發現與貢獻。 1. 於本實驗所使用的搭配方法中，平均運算時間耗時最久的組合為ChiMerge + DROP3 + KNN，其次為DROP3 + ChiMerge + KNN；平均運算時間耗時最短 的組合為IB3 + MDLP + KNN，其次為DROP3 + MDLP + KNN。整體而言， 先執行離散化再執行樣本選取之組合所花費的平均運算時間，會高於先執行 樣本選取後執行離散化之組合所花費的平均運算時間。 2. 本實驗IS + D + KNN 架構下，平均正確率前兩名的組合分別為DROP3 + ChiMerge + KNN 與DROP3 + MDLP + KNN，這兩種組合所得到的平均正確 率，皆較單一離散化過後所得到的平均正確率有所提升，同時這兩種搭配方 法與單一離散化經顯著性檢定後，證實擁有顯著差異；而DROP3 + ChiMerge + KNN 與DROP3 + MDLP + KNN 之間並無顯著差異，故建議可將時間成本 納入考量，選用花費時間較少的搭配方式：DROP3 + MDLP + KNN。先執行 DROP3 樣本選取再執行MDLP 離散化，其組合的平均正確率可達85.11%， 相較於單一MDLP 離散化處理所得到的平均正確率，可以提升大約0.8%。 3. 針對二分類資料集，本實驗結果顯示，在IS + D + KNN 架構下，平均AUC 指標前兩名的組合分別為IB3 + MDLP + KNN 與DROP3 + MDLP + KNN，這 兩種組合所得到的平均AUC，皆較單一離散化過後所得到的平均AUC 有所 提升，同時，經過顯著性檢定後，此兩種搭配方式皆與Baseline 有顯著差異； 然而IB3 + MDLP + KNN 與DROP3 + MDLP + KNN 之間並無顯著差異，故 雖然IB3 + MDLP + KNN 所得到之平均AUC 較DROP3 + MDLP + KNN 高， 仍建議將時間成本納入考量，依據使用情境選用最適當的搭配方式。綜觀以 上所述，在處理二分類資料集的情況下，推薦先執行DROP3 樣本選取再執行 MDLP 離散化進行資料前處理，其平均AUC 可達82.28%，並相較單一MDLP 離散化處理所得到之平均AUC，可以略為提升大約0.8%。 4. 針對多分類資料集進行討論，在本實驗的結果中，平均Kappa 前兩名的組合 分別為DROP3 + MDLP + KNN 與MDLP + DROP3 + KNN，其平均Kappa 分 別可達70.96%與70.11%，並分別相較於單一離散化所得到之平均Kappa，分 別上升約2%與1.1%，由此可以推斷，本實驗所提出的架構使用於多分類資 料集可以讓其平均Kappa 值有提升的趨勢。 5. 從本實驗的結果可以看出，整體而言，先執行樣本選取後執行離散化（IS + D + KNN）所得到的平均正確率，會較先執行離散化後執行樣本選取（D + IS + KNN）來得略高；針對二分類資料集而言，先執行樣本選取後執行離散化（IS + D + KNN）所得到的平均AUC，會較先執行離散化後執行樣本選取（D + IS + KNN）來得高；而針對多分類資料集而言，則是可以由整體平均Kappa 看 出，先執行離散化後執行樣本選取（D + IS + KNN）所得到的平均Kappa 較 先執行樣本選取後執行離散化（IS + D + KNN）來得高。因此，在不同使用 情境下，我們可以選用最適當的搭配方法，來進行資料前處理，提升模型評 估指標。
