本研究延續謝昀峻（2018）的研究以等分法結合技術面策略進行分析，但重構期 原有系統架構，可套用至美股，並建構一個具可擴充性的投資組合模型，將所有變數 設計成投資組合物件中的類別參數，同時設計另一套自動化流程，可用Python 程式覆 寫程式交易軟體Amibroker 執行分析時所需之apx 檔案，客製化回測參數，且直接由 Python 呼叫Amibroker 做回測，再由Python 做後續選股，因此能夠針對投資組合物件 自動進行技術面、基本面、資金使用方法、最大持有股數等變數做績效回測及結果分 析，未來若需增加項目或功能，例如新增布林通道為技術面策略，只需至MultiPortflio 此物件修改參數，後續回測及分析流程皆可自動化執行。 另外本研究也將視覺化呈現分析的功能包裝成模組，使用者能利用Python 之 Import 功能匯入此模組，按照說明設定參數即可看到各種分析結果圖表，比起過往回 測結果皆為大量數據，相信此方法能讓使用者更加了解分析結果。 而經過本研究針對紐約證券交易所股票進行回測之結果，發現當使用凱特通道搭 配最佳化參數時，排名與獲利間相關性會降低，有可能是各檔股票都經過最佳化，能 回測出最高獲利，因此降低了相關性。 另外也發現價格動量因子（52 周價格範圍、7 個月價格變動率、88 個月價格變動 率等）不論在買入持有或凱特通道搭配不同參數及資金用法設定下，皆能有較高的獲 利，其中又以固定參數最為穩定。
