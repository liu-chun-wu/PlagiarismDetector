SOM 神經網路架構如圖 2.4 所示，主要可以分為：輸入層、權值向量、輸出層。 輸入層的神經元間相互獨立，數量等同於輸入特徵的維度並且各自代表每一維的輸入 特徵；權值向量為每個輸出層神經元與輸入層神經元之間相互連結關係的權重，權值 向量中存有與輸入層維度相同數量的權值，在網路訓練完成後，依據SOM 的特性，鄰 近的神經元會具有相似的權值向量；輸出層用來表現資料的拓樸特性，通常以矩陣的 方式呈現在一維或二維空間中，且訓練的過程中會依據輸入的特徵向量，爭取調整權 值向量的機會，並影響鄰近區域輸出層神經元的的權值向量。 根據前段所述架構，SOM 的輸出層神經元會依據輸入層的特徵向量自我組織出有 意義的拓樸結構圖，其拓樸結構可以反映出輸入特徵資料的分佈，故稱之為自組織圖 神經網路。 Input Layer Input Pattern Winner Neuron Output+1 Input Layer 圖 2.4 SOM 神經網路結構示意圖
