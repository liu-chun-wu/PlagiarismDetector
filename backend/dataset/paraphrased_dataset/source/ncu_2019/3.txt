根據以上問題，本研究採用以下的研究步驟、研究設計、驗證與分析方法。 A. 設計預測時間序列的深度學習模型並各自產生預測結果：由於時間序列資料存在前 後關係，本研究必須採用根據時間序列所反映出來的發展過程、方向和趨勢，進行 類推之模型進行股價預測。在模型的使用上，Python 皆有開源模組可以使用，因此 本研究採用建立於Tensorflow 2.0 framework 上的Keras api 進行設計與開發。 B. 設計共通模型：過去預測不同投資標的收盤價的深度學習模型有著顯著差異，故本 研究設計共通性模型，但根據投資標的特性不同進行調整參數，由於不同投資標的 之間有著不同的走勢趨勢，但受限於模型設計必須有著統一的架構，因此本系統將 會設計共同模型，根據不同投資標的的特性調整參數，針對不同投資標的找出最佳 參數以取得最好之預測結果。 C. 設計新的驗證方法，利用「交易策略」進行驗證：過去研究的驗證方法多停留於傳 統統計量，未應用在實際交易，為了檢驗共通性模型是否能於實際交易中獲利，本 研究設計新的驗證方法，利用「交易策略」進行驗證。 貳. 文獻探討 本研究在探討TCN 與加上殘差連接(residual connection)的LSTM 網路何者能在實 際股票投資時產生較高的投資績效，將於2.1 小節介紹相關文獻簡介、2.2 小節介紹 LSTM、2.3 小節介紹TCN、2.4 小節介紹啟動函數、2.5 小節介紹損失函數、2.6 介紹 本研究使用的程式交易、2.7 小節介紹本研究選用的投資標的。
