在建立Pure Neural Network Model 研究架構後，本研究希望由除了模型參數調整以 外的架構調整增進整體架構的準確度，因此由現有數學模型的建構流程(圖3-4)，在人 為挑選參數部份，為保留過往經驗的價值，將人為挑選的參數在資料前處理後分離並訓 練Rule Based Improved Neural Network Model，在預測後同樣將預測結果反標準化並作 為最終預測結果的部份參考，因此將研究架構修改如圖3-5，在資料前處理後由所有資 料訓練Pure Neural Network Model 和從所有資料中挑出人工選擇參數訓練Rule Based Improved Neural Network Model 並各自將預測結果反標準化後分別由權重𝛼與1-𝛼結合 成預測結果，𝛼為人工調整的權重。 圖3-4 現有數學模型建立流程(圖片來源：本研究整理) 圖3-5 Pure NN Model 結合Rule Based Improved NN Model 研究架構 (圖片來源：本研究整理)
