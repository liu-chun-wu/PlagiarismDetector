醫療院所每月將門診、住院等就醫病人的健保醫療費用彙整，以XML 格式的電 子檔於次月20 日以前向健保署申請核銷。申報的資料中包含3 個區塊，分別為總表、 點數清單與醫令清單（健保署，民108），本研究以住院申報的點數清單與醫令清單中 的項目作為資料集樣本來源。首先點數清單為申報案件的摘要整理，彙整病人基本資 料、住院來源、該次住院起迄、出院診斷與住院中的檢查、治療處置及手術的分類代 碼、出院科別、出院醫師、各類別醫療費用統計、Tw-DRG 碼、Tw-DRG 案件支付型 態及申報案件出院動向等多達112 個欄位，欄位標籤以”d”標示，節錄研究採用欄位 說明（表6）。醫令清單為病人該次住院申報醫療項目明細，欄位標籤以”p”標示，欄 位項目有：醫令類別、醫令代碼、病床號、藥品使用的次劑量、頻率、用法與作用部 位等、醫令支付的成數、醫令執行之起迄日期時間、特定治療處置的部位以及器官捐 贈訊息等等有26 個欄位，同樣節錄研究採用欄位說明（表7）。點數清單與醫令清單 欄位屬性”9”為數值型態，”X”為文字型態。 表 6 點數清單-節錄研究資料集採用欄位 欄位 ID 資料名稱 長 度 屬 性 中文名稱/資料說明 d1 案件分類 X 區分健保給付案件代碼（如：一般案件、Tw- DRGs 案件及安寧療護案件等十餘種） d2 流水編號 院所當月申報之流水號。 d3 身分證統一編號 X 外籍人士以居留證或護照號碼填報。 d4 部分負擔代號 X 標註應部分負擔或免部分負擔之代碼。 d6 出生年月日 X 填報民國出生年月日 d10 入院年月日 X 若切帳申報，此欄位填報原始入院日 d11 出院年月日 X 未出院之切帳申報案件，此欄位空白 d12 申報期間-起 X 標明該筆案件流水號申報費用的日期區間 d13 申報期間-迄 X d18 Tw-DRG 碼 X 依據Tw-DRGs 分類架構及原則填報；原則上所有 案件皆必須填報，切帳案件因尚未出院得不填報 d24 轉歸代碼 X 該筆申報案件病人去向或出院後去向。 d25 主診斷 X 住院主要原因，以國際疾病分類代碼ICD-10 CM 編碼申報。 d101 依附就醫新生 兒胞胎註記 X 出生天數≤60 日未取得健保身分之新生兒的性別 與出生胎次，以英文字母表示：大寫為男生、小 寫為女生，英文字母順序為新生兒出生順序 d102 不適用Tw-DRGs 案件特殊註記 X 標示該申報案件是否為Tw-DRGs 案件與不為 Tw-DRGs 案件原因 d103 姓名 X 病人姓名 資料來源：本研究整理，健保署申報格式 https://www.nhi.gov.tw/Content_List.aspx?n=307FADCF95D588EF&topn=D39E2B72B0 BDFA15 表 7 醫令清單-節錄研究資料集採用欄位 資料來源：本研究整理，健保署申報格式 https://www.nhi.gov.tw/Content_List.aspx?n=307FADCF95D588EF&topn=D39E2B72B 0BDFA15
