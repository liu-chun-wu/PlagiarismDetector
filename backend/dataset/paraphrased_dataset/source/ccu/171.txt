行動支付以技術為分類，又可分為「遠端支付」和「近端支付」兩種不同形式。 1. 遠端支付 指透過手機SMS(Short Message Service)或安裝於用戶端 APP 等連接無線網路， 進行消費支付、加值、轉帳等功能服務，屬於連線交易作業模式。遠端支付的作業模 式是不需將手機或行動裝置靠近任何感應器、讀卡機，就可以成功完成支付作業。目 前金融機構提供的各項繳費及轉帳等網路銀行App 模式、第三方支付業務提供之行 動購物模式或由國際發卡機構主導之各類網路服務業務等，皆屬於遠端支付作業模式。 2. 近端支付 主要以手機為支付工具，透過實體商店端末網路，以連線或離線模式完成交易處 理、消費款項支付，作業模式是將手機等支付工具靠近資料讀取設備進行感應，來代 替刷卡進行消費。目前最常見的近端支付技術為近距離無線通訊(NFC)技術及行動條 碼(Quick Response Code, QR Code)技術。NFC 技術是透過無線電波短距離的通訊技 術，讓電子設備與讀卡機之間進行非接觸式，點對點的資料傳輸。QR Code 為矩陣式 黑白相間的點狀及條狀圖形的融合，能表示文字、圖形、聲音等資訊，此技術有容量 大、可靠性高、資料保密防偽、修改資料簡便等優點。
