由於消費者對智慧型手機依賴性不斷的增加，因此，許多業者或銀行開發各種支 付程式並允許消費者透過手機隨時隨地進行交易。台灣的消費市場，行動支付的方式 影響使用行動支付意圖之因素研究-整合科技接受模式觀點 可分為感應式及掃條碼兩種，目前以感應方式為大宗。 (1) QR Code： QR Code 最早應用在貨品的追蹤管理上，它是一種正方形的二維條碼， 用來取代傳統的1D 條碼，可直接將文字資訊儲存在條碼中，並方便資料進 行交換，廣泛使用在亞洲及歐洲。2012 年，萬事達國際組織推出一款行動支 付應用程式「QkR」，消費者可使用該應用程式，輸入卡片資訊並完成身分驗 證後，以掃描QR Code 方式選購食物或其他商品，這一套系統測試於洋基球 場，並在澳洲開始使用。 QR Code 支付的優點包含不限智慧型手機機種、商家硬體建置成本低廉 及使用功能多元等，再加上支付寶、微信支付的成功案例的因素，因此，國 際發卡組織積極推動EMVCo 標準以加速QR Code 統一規格，未來有機會超 越NFC 感應支付，成為行動支付主流付款方式，根據統計，台灣行動支付使 用率前三名，就有Line Pay 及街口支付使用QR Code 掃描方式(資策會產業 情報研究所，民 107)，另外，根據萬事達卡2018 年網購調查指出，台灣有 近75%對科技熟稔的消費者樂於使用QR Code 進行支付。 (2) NFC： 近場通訊技術(NFC)又稱近距離高頻通訊，是通訊協定其中一種，允許兩 個電子裝置在短距離(幾公分)之內進行點對點通訊，NFC 由RFID 射頻識別 技術所衍生而來，NFC 行動支付為感應式支付，主要是建立在行動設備(即智 能手機)和銷售點(POS)系統之間的安全信息交換協議上，跟使用行動設備完 成的所有其他支付方法完全不同。一般而言，使用者必須搭配此硬體支援的 行動裝置才能使用NFC 行動支付，交易過程大多數並不需要額外的授權，但 基於安全考量，部分交易在完成前要求消費者輸入密碼或使用生物辨識來取 得授權，支付款項即可由電子錢包、信用卡、銀行帳戶中扣除，通常用於實 體商店或交通設施中使用。 NFC 支付除了透過行動裝置本身內建的NFC 功能外，也可將NFC 感應 影響使用行動支付意圖之因素研究-整合科技接受模式觀點 元件設置在SIM 卡或SD 記憶卡，例如：TSM-SIM 卡、行動X 卡。在技術 方面，則有TSM、HCE、代碼化服務技術，TSM 是指在手機的SIM 卡植入 交易的安全元件，安全控管由第三方平台負責，並整合電信業者和銀行業者 的資料或資訊交流來進行付款，使用者必須更換成專屬的SIM 卡並搭配特定 的信用卡才能使用該技術；HCE 是由Google 推出的行動支付方案，卡片資 訊及金鑰置放在雲端，交易時由雲端支付系統發出一組虛擬卡號與金鑰，經 過驗證手機安全性與合法性後完成感應交易，使用者必需具備有HCE 功能的 手機；代碼化技術是將信用卡卡號以隨機的「代碼(Token)」取代，用來降低 不同的行動裝置共用同一組卡號的風險，由Visa 於2013 年推出，交易過程 中，店家與收單機構皆不會知道真實的卡號，增加消費者交易安全性並減少 被盜刷的風險，Samsung Pay、Apple Pay、Android Pay 皆屬此類型，其優點 為使用快速方便，配合生物辨識技術，使用體驗更佳。根據金管會2018 年的 統計，國際三大行動支付綁卡數約403 萬張，交易金額累計為215.9 億元。
