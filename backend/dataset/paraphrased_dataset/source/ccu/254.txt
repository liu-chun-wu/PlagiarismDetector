透過大數據環境來處理從電子期刊網站上所蒐集之資料，建置Apache Hadoop 叢 集環境，搭配Apache Spark 為其運算中心。 本研究於Windows 10 專業版作業系統環境下安裝Oracle VM Virtual Box 虛擬化 軟體，透過Oracle VM Virtual Box 虛擬化軟體建置Apache Hadoop 叢集環境，搭配 Apache Spark 的Standalone 運算模式進行資料處理，建立一個Master 虛擬化主機，並 搭配三個Slave 虛擬化主機，Master 虛擬化主機和各個Slave 虛擬化主機透過Secure Shell 安全外殼協定，促使Master 與Slave 虛擬化主機能於網路中建立安全連結。詳 細環境配置請參閱表 5 虛擬環境配置。 表 5 虛擬環境配置 Apache Hadoop 版本 Hadoop 2.6.4 Apache Spark 版本 Spark-2.0.0-bin-hadoop2.6 Java 版本 JDK 7u201 主機名稱 作 業 系 統 CPU 核心數 記憶體 儲存空間 Master Ubuntu 14.04 LTS 64 位元 2 核心 8G 40G Data1 Ubuntu 14.04 LTS 64 位元 1 核心 4G 20G Data2 Ubuntu 14.04 LTS 64 位元 1 核心 4G 20G Data3 Ubuntu 14.04 LTS 64 位元 1 核心 4G 20G 透過大數據與資料視覺化建構學術期刊於資訊安全研究演進
