本研究整理出目前主流的行動支付軟體，介紹各個軟體開發公司、付款方式及流 程，表2-1、2-2 做簡單的統整。 1. LINE Pay LINE Pay 是內建於通訊軟體LINE 的行動支付服務功能，主要是以期在通訊軟 體上數千萬用戶的優勢，從網購出發，並輔以LINE Point 點數回饋，來吸引大量會 員。使用者只要下載「LINE」app，註冊信用卡卡片、進行驗證成功後即設定完成。 付款時只需開啟app 通過驗證後，出示條碼或QR code 給店家進行掃描後即交易完 成。 以創新擴散理論探討消費者使用行動支付行為意圖之研究 2. Apple Pay Apple Pay 是蘋果公司的行動支付和電子錢包服務功能，整合於Apple Wallet 應 用程式中，只能使用蘋果公司推出的iPhone、Apple Watch 和iPad 等行動裝置進行付 款。使用者只要打開「錢包」app，掃描要綁定的信用卡，並依照銀行指示填寫相關 資料後即成功將卡片加入錢包。付款時只需將手指放在Touch ID 上面或按下兩側按 鈕、看一眼螢幕，並將iPhone 靠近讀卡機螢幕，即可輕鬆完成交易。 3. 街口支付 街口支付是街口網絡股份有限公司推出的行動支付平台，主打在地化策略，從現 下的夜市攤販出發，先打入饒河街、師大夜市、寧夏夜市等知名夜市，再打入飲料店 等實體店面。除了支付功能外，街口支付還取得了電子支付的牌照，額外多了儲值、 轉帳等服務功能。使用者只要下載「街口支付」app，註冊會員、綁定信用卡、設定付 款密碼、輸入卡號資料、進行實名驗證成功後即可完成綁定。付款時只需開啟app 輸 入付款密碼後，出示條碼或QR code 給店家進行掃描後即交易完成。 4. Google Pay Google Pay 是Google 推出的行動支付服務，在2018 年一月初，Google 將Android Pay 和Google 電子錢包都整合在Google Pay 服務中。使用者必須使用Android KitKat
