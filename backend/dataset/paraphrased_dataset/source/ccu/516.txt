資訊的定義:資訊泛指數據、事實、說明…等，藉由不同方式存在，且資料經過 整理、分析及彙整，且具備意義和須被妥善保護的重要資產（張正宏,民 101）。在教 育部重編國語辭典網路版（民106）對資訊的定義:「電腦上是指對使用者有用的資 料和訊息的總稱，和未經處理過的資料。」(教育部重編國語辭典網路版)在國家教育 研究院雙語詞彙、學術名詞暨辭書資訊網對資訊的定義:「1. 依據應用資料規則，而 賦予該資料意義。2. 數據、訊息中包含的意義，不隨裝載的物理設備形式變化而改 變。3. 對判斷、選擇、預測、規劃和未來行動進行決策的人有用的訊息。」(國家教 育研究院雙語詞彙、學術名詞暨辭書資訊網)參考ISO/IEC 27002「資訊技術-安全技 術-資訊安全管理之作業規範標準」對資訊的定義是:資訊是一種資產，如同企業對其 他有形資產，對企業而言是不可或缺，因此需加以保護(ISO/IEC 27002,2013)。 綜合以上訊息，可以將資訊定義歸納如下:資訊可以以不同方式存在，經由資料 的整理、分析和彙整出來，且具備意義需被妥善保護的重要資產。 資訊安全定義可追朔最早到1984 年，IBM 的Data Security Support Programs，並 對資訊安全的解釋是「資訊財產不管在什麼樣的情況下，未經授權公開、修改破壞或 使防護系統失效…等等行為」(IBM Data Security Support Programs,1984)，依據 NIST(美國國家標準技術研究所)其將「資訊安全(Information Security)」定義為「保護 訊息及訊息系統免受未經授權的訪問、使用、披露、破壞、修改及銷毀，以提供機密 性、完整性和可用性」(NIST)，參考ISO/IEC 27001:2013 文件中對於資訊安全的見解 則是:「保存資訊機密性、完整性、可用性；涉及如鑑別性、可歸責性和可靠度等等 的性質。」(ISO/IEC 27001,2013)。 表1：個學者對資訊安全的定義彙總表 出處 定義 IBM Data Security Support Programs 對資訊財產在任何情況下，未經授權公開、修改破壞或 使防護系統失效…等等。 ISO/IEC 27001:2013 保存資訊機密性、完整性和可用性；也能涉及鑑別性、 可歸責性和可靠度等性質。 ISO/IEC 17799(2001) 資訊是具有價值的重要商業資產，需要加以保護，以免 受各種威脅攻擊而維持組織營運，讓可能發生的損失降 到最低。 葉乃菁、李順仁(2004) 狹義來說，資訊安全是指維護儲存和使用之安全性 ; 廣義來說，資訊安全是保護資訊的產生、儲存使用傳送 展示及控制等來源，防止非法存取竄改或惡意損毀和破 壞。 Deborah Russell,G.T.Gangemi(1992) 資訊安全是保護任何與電腦有關的事務，管理程序與安 防護應用在系統及硬體以及資料中。 Hutt(1995) 資訊安全威脅分成:潛在威脅、實體的災難、裝置故障、 軟體故障、人員操作錯誤、資料誤用和遺失等威脅 資料來源:本研究整理 綜合以上文獻，總結資訊安全定義如下:資訊安全是為了預防天災、人為或是無 心破壞，保護系統和資料的總總用途，達成未經授權的存取或是變更並確保合法使用 者對資料可用性的目的。資訊安全的範圍同時涵蓋技術層面和管理層面，缺一不可。
