為了加強物件在大小、位置、及旋轉角度上的推論能力，我們以 9DoF ADM-YOLO 為基礎加入原始深度資料分支來增強空間訊息的獲取能力， 我們做了三種不同的分支設計，第一種是將原始深度資料直接以雙線性 內插降維至輸出各輸出層的解析度，再並聯 (concatenate) 骨幹特徵，最 後直接以感興趣區塊卷積提取特徵，如圖 4.4 紅色線段所示，我們稱為 9DoF ADM-YOLO + depth_ bilinear。此分支實驗的資料共有20 類20000 張資料，其中18000 張做為訓練資料，2000 張做為測試資料，訓練200 回合，實驗結果如表 4.3 所示，9DoF ADM-YOLO 與以其為基礎加入雙線 性內插降維的分支網路之結果比較。其中 9DoF ADM77-YOLO 表示採用 77 個採樣點的感興趣區塊卷積核之 9DoF ADM-YOLO 架構。 表 4.3. 加入雙線性內插降維的分支網路之各項改動對準確度的影響。 架構 ADM 7x7 ADM 13x13 以雙線性內插 降維的分支網路 mAP 9DoF ADM-YOLO ✓ 96.54 ✓ 96.90 9DoF ADM- YOLO + depth_ bilinear ✓ ✓ 95.73 ✓ ✓ 96.12 由上表可得知若將原始深度資料逕直以雙線性內插降維，將會導致 網路效能下降。推測其原因應該是由於將原始影像以雙線性內插從 416416 解析度，直接降維至輸出層的1313、2626、及5252 解析度 會損失相當多的資訊；有鑑於此，我們改為使用全程步伐 (stride) 為2 的 33 卷積代替雙線性內插來降維，過程中不使用任何步伐為1 的卷積，目 的在使該分支輸出的特徵為低階特徵 (low level feature)，以保留更豐富的 - 50 - 空間訊息。最後，將分支輸出的特徵並聯骨幹特徵後直接以感興趣區塊卷積提取特 徵，如 圖 4.5 紅色線段所示。為了顧及實驗的速度及效率，此實驗只使用 10 個物件類別進行，類別分別是──調味罐、餅乾盒、糖盒、番茄罐、芥 末瓶、魚罐頭、布丁盒、果凍盒、肉罐頭、及香蕉，共10000 張資料，其 中9000 張做為訓練資料，1000 張做為測試資料，網路訓練共200 回合， 實驗結果如表 4.4 所示。 表 4.4. 加入全程步伐為2 的33 卷積降維的分支網路之各項改動對準 確度的影響。 架構 ADM 7x7 ADM 11x11 ADM 13x13 全程以步伐為2 的33 卷積降維 的分支網路 mAP 9DoF ADM-YOLO ✓ 97.46 ✓ 98.89 9DoF ADM-YOLO + depth_conv/2_concat ✓ ✓ 99.49 ✓ ✓ 99.64 由上表可得知在 9DoF ADM77-YOLO 內加入以33 卷積降維的分 支網路可以使網路得到2.03%的 mAP 提升，得到新架構 9DoF ADM- YOLO + depth_conv/2_concat，再將感興趣區塊卷積核之採樣點數增加為 1111，可再使網路提升0.15%的 mAP。從實驗中可看出該分支架構能有 效協助網路推論，因此我們以該架構進行20 類的實驗，但經過實驗後得 到 mAP 為95.73%的結果，比起沒有加入該分支網路的 9DoF ADM- YOLO 結果下降了0.81%的 mAP，如表 4.5 的實驗結果所示；推測其原 因是因為遇到的問題變複雜了，因此我們在並聯骨幹特徵後再增加一層 步伐為1 的33 卷積來提取特徵，之後交由感興趣區塊卷積提取特徵， 如圖 3.8 紅色線段所示，該架構稱為 9DoF ADM-YOLO + depth_conv/2_concat_conv。此實驗共有20 類20000 張資料，其中18000 - 51 - 張做為訓練資料，2000 張做為測試資料，訓練200 回合，實驗結果如表
