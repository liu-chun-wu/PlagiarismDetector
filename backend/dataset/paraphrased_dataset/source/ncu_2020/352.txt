在本實驗中，共有20 類20000 張資料，其中18000 張做為訓練資料， 2000 張做為測試資料，網路訓練共200 回合，我們在 9DoF SE-YOLO 內 加入對齊偵測模組並將網路骨幹中的SE 模組去除，新的架構我們稱為 9DoF ADM-YOLO。我們的實驗改變了感興趣區塊卷積之卷積核 (kernel)， 使其採樣點數目變更為77 和1313，實驗結果如表 4.2 所示。 表 4.2. 加入對齊偵測模組之各項改動對準確度的影響。 架構 ADM 7x7 SE 模組 ADM 13x13 mAP 9DoF SE-YOLO ✓ ✓ 93.59 9DoF ADM-YOLO ✓ ✓ 93.81 ✓ 96.54 ✓ 96.90 從實驗中能看出，在尚未將 SE 模組從網路骨幹中去除掉的 9DoF SE-YOLO 中加入對齊偵測模組，使得 mAP 得到了0.47% 的提升；之後 我們將 SE 模組去掉後 mAP 又得到了2.73% 的提升，由此我們觀察到 加入 SE 模組並不利於我們的卷積神經網路推論，而新架構我們稱之為 9DoF ADM-YOLO。最後為了能更多的獲得特徵訊息，我們將感興趣區塊 卷積的採樣點從77 提升到1313，這個改動使得 mAP 能提升0.43%。 - 49 -
