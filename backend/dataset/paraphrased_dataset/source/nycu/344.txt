本節將列述幾篇將深度學習應用於多目標追蹤的文獻，且這些文獻都是以 tracking-by-detection 為主。Bewley 等人於[12]中時提出以Kalman Filter 預測幀與 幀之間的bounding box 資訊，其中包含中心座標及尺寸大小。並以Hungarian Algorithm 進行圖像配對。其cost 矩陣為使用Kalman Filter 預測框與新偵測之間 的IOU，低於某門檻值的預測框與新偵測將不進行配對。這篇論文專注於短時間 的目標追蹤，若發生遮蔽情況，則ID 切換情況較為頻繁。文獻[13]發表於2017， 目的為改善[12]的缺點，加入了圖片外貌因素進行追蹤。他們使用CNN 模型擷 取偵測框的特徵並加入配對演算法。將ID switches 減少了45%。此方法顯示使 用CNN 擷取之特徵在追蹤的問題上，能夠有效的辨識出新的偵測框屬於的是哪 一個追蹤人物軌跡。 Leal-Taixé 於[11]提出使用兩個階段的學習追蹤行人。第一階段使用Siamese 架構判斷前後幀之偵測框是否為同一人，其CNN 的輸入為像素值以及光流資訊。 作者在文中探討了幾種Siamese 架構的輸入，發現將兩張圖像堆疊在一起作為一 個輸入的效果最好。第二階段使用相對位置，速度等資訊，結合第一階段得到的 兩圖片間的相似度，一起作為gradient boosting classifier 的輸入，得到最終配對 預測。 Xiang 於文獻[14]使用Markov Decision Process 的決策方式進行追蹤。並在 決策過程中使用增強式學習，當追蹤過程中與ground truth 不同時（錯接或漏接） 則更新其policy。在tracked state 下使用光流資訊，計算Forward-backward error 來評估追蹤的穩定性。並加上過去幾幀與新偵測的平均overlap 作為另一個評估 指標，目的在於解決追蹤固定在不會移動的錯誤偵測上（例如背景）。在圖像關 聯性計算上，也是使用Hungarian Algorithm 但搭配強化學習得到的權重進行分 配。此方法使用MDP 處理追蹤軌跡的不同狀態，如圖2-1，若狀態轉換的規則 處理得當，對於追蹤軌跡的處理也較為明確。 Zhu 等人於[4]首先採用現有的單目標追蹤器，ECO tracker（Danelljan 等人 於[15]提出），在圖像匹配中使用兩個神經網路，Spatial Attention Network（SAN） 和Temporal Attention Network（TAN）。在SAN 使用Siamese 架構，分別擷取追 蹤序列和偵測框的特徵圖，生成attention map 後換算成權重覆蓋上原特徵圖，使 得空間資訊能專注在兩個圖像間較為相似的區域。再將此特徵圖輸入TAN 網路， 此網路最終會生成一個匹配分數，用來表示此偵測是否接在與其比較之追蹤序列 上。
