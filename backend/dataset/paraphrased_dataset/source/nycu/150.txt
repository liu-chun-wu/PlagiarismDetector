個公開的胸腔X 光片資料集ChestX-ray14，此資料集中包含了超過十萬張的匿名影像及 對應的數據。此資料集中含有14 種不同病症的胸腔X 光片，分別為：肺不張(Atelectasis)、 心臟肥大(Cardiomegaly)、肺積液(Effusion)、浸潤(Infiltration)、腫塊(Mass)、結節(Nodule)、 肺炎(Pneumonia)、氣胸(Pneumothorax)、肺實變(Consolidation)、肺水腫(Edema)、肺氣腫 (Emphysema)、纖維化(Fibrosis)、胸膜增厚(Pleural Thickening)、疝氣(Hernia)，共有112,120 張影像。 資料集中的影像，是來自NIH 臨床中心的影像擷取及傳輸系統(Picture Archiving and Communication System)中所擷取，共含有112,120 張大小為1024 1024 像素的灰階胸腔 正面X 光線影像，約為NIH 中60%的胸腔正面X 光片影像，其中包括了30,805 位患者 的影像資料。在元資料(Metadata)的部分則包含了影像編號(Image Index)、發現(異常)標 籤(Finding Labels)、追蹤編號(Follow-up #)、病患編號(Patient ID)、病患年齡(Patient Age)、 病患性別(Patient Gender)、拍攝位置(View Position)、原始影像寬長(Original Image[Width, Height])、原始影像像素間距(Original Image Pixel Spacing[x, y])的各項資料。表三為 Metadata 中各項屬性欄位對應表。 表三、ChestX-ray14 資料集所有欄位之各項屬性欄位對應表 欄位名稱 英文名稱 欄位名稱 英文名稱 影像編號 Image Index 發現(異常)標籤 Finding Labels 追蹤編號 Follow-up # 病患編號 Patient ID 病患年齡 Patient Age 病患性別 Patient Gender 拍攝位置 View Position 原始影像寬 Original Image[Width] 原始影像長 Original Image[Height] 原始影像像素間 距 Original Image Pixel Spacing[x, y] 本論文使用NIH 發布的公開資料集ChestX-ray14 作為訓練資料，在過往的研究中， 有些論文也使用相同的公開資料集作為訓練資料，本論文將依序整理其中三篇使用相同 資料集的研究論文、分析方法與結果。
