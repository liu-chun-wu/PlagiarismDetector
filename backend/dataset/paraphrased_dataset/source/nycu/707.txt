圖 19 模型 - Cross validation 交叉驗證，提升預估模型的效能。在機器學習訓練模型的過程中，使用同一 資料集去做函數學習並預測，這樣訓練出來的模型因為過程中只是重複剛剛看到 的樣本標籤容易會有一個完美的分數，這樣運作模型預測出來的數據不見得有參 考價值，此種狀況我們稱之為：Overfitting，為了避免資料和模型之間的 overfitting 關係，可採用cross validation 將資料集分成六群，而每群中分成 兩個部份：X_test、y_test 做訓練和驗證使用，一直重覆的訓練、測試進行六次 - Training set、Testing set Training set 稱之為訓練數據、Testing set 稱之為測試數據，在機 器學習的過程中，為了驗證訓練出來的模型是否成功，會將資料集切分成 兩個部份，一部分做為訓練使用：Training set、一部分使用訓練模型做 預估，這樣的資料為：Testing set。接下來的實驗過程中，會採取 Training set：70%、Testing set：30%。
