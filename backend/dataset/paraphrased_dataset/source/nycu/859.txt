本實驗想要證明出這套演算法較佳，因此在實驗中想要藉由標準人對每部電影的未 調整分數(std)、非標準人對每部電影未調整過分數(unstd_unadj)及非標準人對每部電影 調整過的分數(unstd_adj)做比較，若標準人跟非標準人對每部電影調整過的電影分數差 距小於標準人跟非標準人對每部電影未調整過的電影分數差距，也就是(𝑠𝑡𝑑− 𝑢𝑛𝑠𝑡𝑑_𝑎𝑑𝑗) < (𝑠𝑡𝑑−𝑢𝑛𝑠𝑡𝑑_𝑢𝑛𝑎𝑑𝑗)，則代表分數更接近真實值，經過評分優化系統算出 來的效果更好，在實驗中會算出總Error 值及評分調整過改進的比例。 而在實驗中，將會調整兩種變數去做深入的探討： a. 標準人對電影的評分是否調整： 因為標準人所評出的分數又有分調整和未調整兩種狀態，因此實驗中這個 參數的調整將又會分為兩組子實驗，如下圖3-5。 第一組 ( 標準人調整電影分數 − 非標準人調整電影分數 ) ( 標準人調整分數 – 非標準人未調整電影分數 ) 第二組 ( 標準人未調整電影分數 – 非標準人調整電影分數 ) ( 標準人未調整電影分數− 非標準人未調整電影分數 ) 圖3-5 探討此評分優化系統是否有效的實驗步驟 b. 計算每位評分人的信用評分時使用不同的標準： 在此部分實驗中有一個最重要的部分就是要計算每位評分人的信用評等， 而在3.5 中裡面有提過信用評等的計算方法，在公式中有一個變數α，主要是代 入評分人信評的標準，由於我們採用的是統計分布方法，也就是藉由每筆評分 分數分布的地方來判斷評分人是否評分資料公正並計算總分及差異度，根據統 計學中的信賴區間，已知中心為平均值，則在1 倍標準差內會涵蓋約68%的資 料，2 倍標準差內會涵蓋約95%的資料，則3 倍標準差內會涵蓋約99.7%的資 料，也就是說，到3 倍標準差幾乎已經涵蓋所有的資料。從以上的理論基礎， 在本實驗中的α也就代入1、2 和3 做探討。 總結上述的實驗，最重要的核心的就是要找到標準人，並以標準人為基準，將非標 準人的分數做調整前及調整後的分數比較，目的就是要探討我們所設計出的評分優化系 統是否有做到評分的精準化。
