Yolo v3 一個是one-stage learning 方法，不需要region proposal 的階段，經過單一次 檢測即可獲得最終的偵測結果，雖然辨識率大多比起two-stage learning 來說較低，但整 體仍是可接受範圍，利用整張影像直接輸入網路架構，直接預測 bounding box 的座標 以及其信心值和所屬的類別，在速度方面仍具有相當大的優勢。在本研究中，由於要將 一張影像切割成八張Virtual PTZ 影像作偵測（3.2.2），所以速度方面不希望太慢，因此 選擇此架構來進行研究。 Yolo v3 採用Darknet-53 來提取特徵，特徵層從單層的13x13 變成了3 種層(13x13、 26x26、52x52)，13x13 層用來檢測大型物件、26x26 檢測中型物件、52x52 檢測小型物 件，利用多尺度的特徵增加特徵的豐富程度，Yolo v3 採用類似於FPN(Feature Pyramid Networks)的Upsampling 和Concatenation 的做法融合了這3 種尺寸，並且在不同特徵層 獨立進行偵測，使得對於小目標偵測的效果有明顯的提升。
