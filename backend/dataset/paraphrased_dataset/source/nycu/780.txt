本研究使用的資料集來自美國國家癌症研究院的Surveillance, Epidemiology, and End Results (SEER)計畫的資料，這個資料集收錄了癌症發病率與死亡等數據。其中包含 了各種種族，涵蓋了全美28%的人口資料，主要欄位有腫瘤原發部位、診斷時的腫瘤型 態、分期資訊的相關數據。資料儲存方式是以TXT 文件檔案儲存，病患紀錄的欄位是 根據資料描述文件所定義的欄位長度進行切割，共分割成137 個欄位，最後本研究留下 30 個欄位，範例如表3-1。 表3-1、欄位資料範例 欄位名稱 資料屬性 資料型態 範例值 Race Unordered String ‘White’ Sex Nominal Integer Stage Ordinal String ‘IIA’ Tumor Size Continuous Float 12.6 Age at Diagnosis Continuous Integer 除了上述的屬性外，資料特徵主要可以分成個人資料、臨床資料、預後資料和治療 資料四大類。個人資料主要包含年齡、種族、性別及婚姻狀態等；臨床資料是診斷後一 些癌症或腫瘤相關的資訊，例如：淋巴結的數量、腫瘤大小、是否有移轉到遠處、腫瘤 的原發位置、腫瘤偏側方向、細胞與腫瘤是否分化或區域淋巴結數量等；預後資料，包 含了癌胚抗原濃度、神經侵入、18q 染色體丟失等相關資訊；治療資料，包含了手術狀 況與手術類型等相關資料，包含了主要部位的手術、局部淋巴結手術的範圍、是否有進 行化學治療、是否有放射治療和治療順序等。在3-4 小節會詳細說明本研究是如何進行 特徵的選擇。
