為了研究關於繪圖的手眼協同控制技巧，受測者被要求在數位(RWI, VRI, VREI)及傳統紙筆的實驗條件下沿著圖片的輪廓描線。受測者需要沿著五種不同的 圖片描邊(魚、人物、烏龜、雨傘及房子)，在這些圖片中，輪廓的長度、曲度及交 叉線的複雜度皆有一定程度的變化。我們比較同一個測驗在不同實驗條件下的結 果。與Writing 的任務採用相似的方式，由於數位實驗條件與紙本實驗條件的解析 度及畫出的線寬有所不同，我們利用影像處理的骨架化的方式，將線寬進行標準 化的處理，以確保不同的條件的結果可以被客觀的比較。我們將不在輪廓線上的 像素設為誤差像素，該像素到輪廓的垂直距離為誤差距離，之後累積所有誤差像 素的誤差距離除以誤差像素的數目以得到平均誤差距離。因此我們將ANOVA 模 型設計成(實驗條件-RWI vs. VRI vs. VREI) X (5 種不同圖片) X (數位設備輸入與 傳統紙筆輸入)，在所有關於參數的測驗皆被證實之後。 1. T3-誤差距離 經由ANOVA 分析之後，我們發現圖片的不同對結果有顯著的差異F(4, 160) = 29.33, p < 0.001, 𝜂2 = 0.42，條件實驗不同對結果有顯著的差異F(2, 40) = 6.93, p = 0.003, 𝜂2 = 0.26，不同輸入設備對結果有顯著性的差異，F(1, 160) = 27.68, p < 0.001, η2 = 0.41，在不同輸入設備條件下，實驗條件的不同對結果具 有顯著性的差異F(2, 160) = 18.79, p < 0.001, η2 = 0.48，在不同圖片之下，輸入 設備的不同對結果具有顯著性的差異F(4, 160) = 4.067, p = 0.004, η2 = 0.092， 在圖片、輸入設備及實驗條件的交互作用項之係數具有顯著性F(8, 160) = 2.05, p = 0.04, η2 = 0.093，在這些互動之中，我們著重於不同實驗條件下，輸入設備不 同的影響。 我們利用Bonferroni 的事後配對分析，發現在平均誤差距離在RWI 的條件 下，數位輸入與紙本輸入的條件中並無顯著的差異。然而在VRI 的條件下，數位 設備輸入條件(M=3.56, SD=0.70)的平均誤差距離是顯著的大於與紙筆輸入比較時 (M=3.13, SD=0.28), p = 0.008。同時，在VREI 的條件下，數位設備輸入條件 (M=3.8, SD=0.56)是顯著的高於與紙筆輸入比較時(M=3.12, SD=0.28), p < 0.001. Tukey’s HSD 的事後配對分析在平均誤差距離的分析上，在RWI、VRI 及VREI 條件的紙筆輸入並沒有顯著的差異，然而在數位設備輸入的平均誤差距離上，VRI (M=3.56, SD=0.65)是顯著高於RWI(M=2.93, SD=0.29)，p = 0.005，VREI (M=3.82, SD=0.47)也相當顯著的高於RWI，p = 0.001，如Fig.17 所示。 Fig. 17 左圖為使用數位設備輸入，右圖為傳統紙筆輸入的平均誤差距離。
