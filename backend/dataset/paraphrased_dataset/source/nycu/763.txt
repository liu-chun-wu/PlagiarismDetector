(1).Node.js〔4〕 Node.js 是一個能夠在伺服器端運行JavaScript 的開放原始碼、跨平台 JavaScript 執行環境。使用事件驅動、非阻塞和非同步輸入輸出模型等技 術來提高效能，可優化應用程式的傳輸量和規模。這些技術通常用於資料 密集的事實應用程式。本研究希望可以透過Node.js 對於大量資料的處理 以及可以對於資料整理並且回覆的非同步需求，這兩個優點來進行開發的 主軸。 (2).MySQL〔5〕 MySQL 是一個關聯式資料庫管理系統，有著效能高、成本低、可靠性。研 究中希望資料可以應用於Line 以外的通訊軟體程式，所以需要透過單一的 資料庫來整合故宮Open Data〔3〕與之相關資料。 (3).JSON〔6〕 JSON（JavaScript Object Notation）是一種由道格拉斯·克羅克福特構想 設計、輕量級的資料交換語言，以文字為基礎，且易於讓人閱讀。研究中 會將Line 的對話以及從網頁中所得到的資料進行彙整與處理成資訊，因此 會進行資料的傳輸與處理，Json 能有效並且進行管理 (4).HeroKu〔7〕 Heroku 是一個支援多種程式語言的雲平台即服務。可以提供免費的伺服器 與資料庫，因此在研究中將聊天機器人的相關模組建立HeroKu 的雲端之 上。 (5).Threejs〔8〕 Threejs 是一個輕量化的JavaScript API，用於網頁瀏覽器上創建與顯示 3D 動畫、圖形向量計算，常與HTML5 中cancas、SVG、WebGL 一併使用， 因此本研究中選用其中數種表現形式作為資料總覽的表達，並妥善利用 API 中所提供的動畫，提升數位展場的吸引力。
