一般輸入法多半採用OnKey 事件來組字，即當點擊某個按鍵時，程式會針對 該按鍵對應的字根去拼出對應的單字，但為了達到錯誤校正，我們必須採用解析度 更高的方法來組字；透過OnTouchListener 中的onTouch Event，我們可以很容易的 取得點擊的位置，用以計算每次點擊的權重。 private View.OnTouchListener handleTouch = new View.OnTouchListener() { @Override public boolean onTouch(View v, MotionEvent event) { int x = (int) event.getX();  //取得點擊的X 座標 int y = (int) event.getY();  //取得點擊的Y 座標 switch (event.getAction()) { case MotionEvent.ACTION_DOWN:  //按下 Log.e("TAG", "touched down"); break; case MotionEvent.ACTION_MOVE:  //滑動 Log.e("TAG", "moving: (" + x + ", " + y + ")"); break; case MotionEvent.ACTION_UP:  //放開 Log.e("TAG", "touched up"); break; } return false; } };
