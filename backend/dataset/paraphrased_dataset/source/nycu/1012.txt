倒傳遞類神經網路(Back-propagation Neural Networks)由McClelland、 Rumelhart、Hinton 和Williams 等學者於1986 年所提出，是一種能適應新環境的 多層監督式學習網路，透過自我的學習來建立處理問題之知識學習模式，具有 分析、預測、分類等能力，如同人類大腦神經網路的學習過程，針對過去的經 驗或資料重覆學習。 相較於傳統的統計方法，倒傳遞類神經網路相對有更強大的運算處理與分 析能力，可以用來處理高維度與非線性的複雜問題，因此近年來被廣泛應用在 各種領域特別是以金融投資決策領域最多。 HARRIS 等學者(1986)[9]以紐約證券交易所(NYSE)普通股市交易資料做檢 定，實證發現開盤時之前四十五分鐘存在顯著的異常報酬及呈現所謂的U 型曲 線(U-Shaped)型態。 WEI 等學者(2012)[19]建構一個複合系統模型，先使用技術指標找出關聯程 度，並用逐步回歸與決策樹的方法提升Recurrent Neural Network (RNN)的預測準 確度。 BARAK 等學者(2015) [1]提出複合式架構的預測模型，比單一模型預測 有較高的準確度。 CHEN 等學者(2013)[6]使用適應性模糊類神經網路架構來預測股價的趨勢 變化，並獲得較高的準確度。HAFEZI等學者(2015)[8]提出智慧化模型 Bat-Neural Network Multi-Agent System (BNNMAS) 是以四階類神經網路結構的學習演算 法進行預測股價，實驗結果呈現很好之績效。 李俊賢(2017)[22]應用灰色模型GM(1,1)動態預測臺灣股市加權指數與櫃買 中心加權股價指數研究提出以灰色理論模型GM(1,1)對臺股加權指數以及上櫃 市場指數做每五分鐘一次以及每30 秒一次的動態預測。驗證結果顯示GM(1,1) 模型在每五分鐘及每30 秒對臺股加權指數的預測效果佳，但對櫃買指數的預測 效果有限，某些情況會出現離群值導致預測結果失真，投資人須得謹慎判斷。 廖玟柔(2017)[27]透過類神經網路結合多面向指標創建模型，以技術面指標、 籌碼面指標、國際股市指標和國際匯率指標進行彙總，以不同的訓練集時間長 短進行實驗，與組合不同面向的指標進行研究，找出預測準確度最高之模型。 陳芊邑(2017)[26]使用S&P500、Nasdaq 、DJIA 三個股價資料，來檢驗 GARCH 模型與類神經網路模型，兩者在股價報酬率上之點預測及區間之預測優 劣。實證結果顯示，採用最高報酬率、最低報酬率為解釋變數時，類神經路模 型績效比較好。 HUANG 等學者(2016)[10]以市場輪廓指標與類神經網路去建構較佳的交易 進場點，提出以5 天的POC 歷史資料去預測最佳的進場點，說明此POC 的指標 資料是買賣雙方都能接受最多價格的成交點。此外對於成交量都是很有參考價 值的指標。從文獻可知金融市場預測是屬於動態的預測模式[1~6,10,14]針對研究 相對低風險的進出場點也是屬於動態預測的範疇，因此使用倒傳遞神經網路相 較於傳統統計方法不止有更佳的準確度，還能解決非線性的複雜問題，更能顯 示其方法的適用性。因此本研究採用倒傳遞神經網路進行預測股價趨勢。 第三章、研究方法
