(1)買方尾部物理量(Single Print Buyer Tail Force)：下端點尾部TPO 價位差乘上 成交量。市場輪廓圖中，價格的極端低點出現的單字母分區。這是一個重要的 參考點，表示一些長線買方熱烈的反應低於價值區間的價位，買方之間的競爭 使價格快速上漲，所以，尾部愈長，反應愈強烈。尾部至少應有兩個以上的單 字，才有意義。運用金融工程物理學的概念:F=M*A 將買方尾部物理力量求出: F= 成交量(M) X 買方尾部價位差(A 加速度)。 (2)賣出尾部物理量(Single Print Selling Tail Force)：上端點尾部TPO 價位差乘 上成交量。運用金融工程物理學的概念:F=M*A 將賣方尾部物理力量求出: F=成 交量(M) X 賣方尾部價位差(A 加速度)。 (3)買方價格幅度擴展物理量(Buyer Range Extension Force)：初期平衡以上價格 區間TPO 價位差乘上成交量。 (4)賣方價格幅度擴展物理量(Seller Range Extension Force)：初期平衡以下價格 區間TPO 價位差乘上成交量。 (5)市場擺動因子(Rotation Factor, RF)：擺動因子是綜合分析市場力量的一個簡 捷的量化工具。如某一時間分區，最高價高於相鄰前一個時間分區的最高價時， 則擺動因子賦值+1。如果最高價低於相鄰前一時間分區的最高價，則擺動因子 賦值-1。反之亦然，如果某一時間分區內最低高於相鄰前一時間分區的最低價， 則擺動因子賦值+1，反之，賦值-1。對每個時間分區的最高價(或最低價)相等， 那麼擺動因子賦值0。對每個時間分區進行同樣的賦值，最終的數字加總為該交 易日的市場情緒。 (6)成交量(Volume)：即每個單位交易時間的價格成交量，以本文為1 分鐘K 線 的價格成交量，用以表達市場上交易的情緒能量。 表2 Take 資料計算範例 日期/時間 開盤 高點 低點 收盤 成交量 平均 成交量 2017/7/3 08:46 2017/7/3 08:47 2017/7/3 08:48 2017/7/3 08:49 2017/7/3 08:50 2017/7/3 08:51 2017/7/3 08:52 2017/7/3 08:53 2017/7/3 08:54 2017/7/3 08:55 以每1 分鐘為一個TPO，構成10 分鐘的市場輪廓圖如圖7，如此將會顯示出 幾處的關鍵資訊，例如：某個TPO 的交易量、Upper Tail 區間以及Lower Tail 區間、上方/下方價格幅度擴展(Upper Range Extension & Lower Range Extension)。 本研究市場輪廓理論中，將控制點上下70%的價格範圍視為價值區間，而價格 幅度及價值區間詳細的計算方式如表2。以2017/7/3 08:55 的Take 資料計算範例 結果如表3 所示。 圖 7 市場輪廓特徵區間圖 Price Volume Count Print 10,274 A <- Value Area 10,273 A 10,272 AC 10,271 ABC 10,270 ABCD 10,269 ABCD 10,268 ABCD <- Point of Control 10,267 ABD 10,266 ABD 10,265 ADE 10,264 ADEF 10,263 ADEF 10,262 ADEF 10,261 ADFH 10,260 ADFGH <- Value Area 10,259 AFGH 10,258 AFGHI 10,257 FGHI 10,256 FHIJ 10,255 HIJ 10,254 IJ 10,253 IJ 賣價格幅度 擴展 賣方尾部 控制點 價值區間 價值區間 表3 市場輪廓指標計算範例 日期/時間 2017/7/3 08:55 一分線 期間 開盤 10,260 收盤 10,256 成交量 價格範圍 10,253~10,274 價格幅度擴展上方 10,274~10,274 初期平衡階段 10,258~10,274 價格幅度擴展下方 10,253~10,258 價格中位數 10,263 價值區間 10,260~10,274 賣方尾部點數 賣方尾部範圍 10,273~10,274 買方價格幅度擴展 控制點 賣方價格幅度擴展 買方尾部範圍 10,253~10,253 買方尾部點數 擾動因子 -5 控制點上方總成交量 控制點下方總成交量 價格區間總成交量 買方尾部物理量 賣方尾部物理量 買方價格幅度擴展物理量 賣方價格幅度擴展物理量
