本研究使用SEER 資料集，取用2004 年至2014 年結腸直腸癌的病患資料，並取用 ICD 代碼為C18 開頭的資料，C18 代表患處在結腸，排除直腸與其他部位，共約24 萬 筆資料。接著在資料特徵選取上，我們主要分成兩個部分，以醫療文獻為基礎的預後因 子特徵、機器學習算法所選擇的特徵。接著針對上述兩個部分的特徵進行存活率模型的 建置，主要模型有1 至5 年的存活率模型與多類別的預測模型；多類別模型將目標值切 割成六等分，分別以12 年為區間，主要使用機器學習方法有隨機森林、極度隨機樹和 整體學習的方法。 欄位前處理 在部分特徵值的資料中有超過半數的缺失值，如神經侵入、環週切緣等，造成缺失 值的原因很多種，有可能是取得標籤的醫療上的成本過高或是登入資料時系統故障等都 有可能，因此在這裡會多一個處理步驟，主要針對這些缺失值過多的欄位進行處理，避 免缺失值資料的過度影響模型的預測，即時能夠準確預測，也會對結果產生誤解，產生 出不如預期的模型。根據3.3 小結的資料前處理與欄位預處理，我們整理出共30 個特徵 欄位，如表4.1。每個特徵的詳細資料我們呈現在附錄二。 表4-1、SEER 資料欄位 中文特徵名 英文特徵名 婚姻狀態 Marital Status 性別 Sex 年齡 Age At Diagnosis 惡性腫瘤數量 Sequence Number--Central 癌症主要部位 Primary Site 組織型態 Histologic Type ICD-O-3 性態碼 Behavior Code ICD-O-3 細胞分化程度 Grade 區域陽性淋巴結數量 Regional Nodes Positive 區域淋巴結總數 Regional Nodes Examined 腫瘤大小 Tumor Size 癌症擴展程度 Extension 淋巴結侵襲程度 Lymph Nodes 癌胚抗原濃度 Carcinoembryonic Antigen (CEA) 局部淋巴結的臨床評估 Clinical Assessment of Regional Lymph Nodes 環週切緣 Circumferential Resection Margin (CRM) 腫瘤狀態 Derived AJCC-6 T 淋巴結轉移 Derived AJCC-6 N 遠處轉移 Derived AJCC-6 M 癌症分期 Derived AJCC-6 Stage Grp 外科手術過程 Rx Summ—Surg Prim Site 淋巴結手術的範圍 Rx Summ—Scope Reg Ln Sur 無進行手術的理由 Reason For No Surgery 種族 Race Recode (W, B, AI, API) 是否西班牙人 Origin Recode Nhia 首顆腫瘤是否為惡性 First Malignant Primary Indicator 神經侵襲程度 Perineural Invasion 原位癌/惡性腫瘤數量 Total Number Of In Situ/Malignant Tumors For Patient 是否執行過化療 Chemotherapy Recode 併發症數量 Comorbidity Cancer Number 總數
