本研究會採用3 種在預測運動賽事勝負上，取得相當不錯的成果的監督式學習的預測 模型，而過往幾乎沒有關於副本勝負預測方陎的文獻，因此根據現有的戰鬥紀錄，本研 究也提出了2 種非常陽春的模型來估測團隊的勝負，並期望3 種機器學習模型的表現能 優於這2 種簡易的估測方法。 1. 機器學習模型 （1）Linear Regression 線性回歸（Linear Regression）是最常見也是最簡單的回歸模型，該模型 會根據資料找出一個線性方程式，用來預測新的資料應該在哪個位置。 每一個資料點與線性方程式的距離平方和稱為誤差，而線性回歸的目標 變是嘗詴將該誤差降到最小，並以此來決定線性方程式的權重。 18 許多玩家除了主要的角色外，還會練一些分身（Alternatives）體驗別的職業或是劇情，相較於主要的 本尊角色，分身在遊玩的時間通常會少上許多，裝備也會有所落差，而透過團隊帶分身角色打副本便可 以快速提高裝備等級、跟上團隊進度。 （2）Support Vector Machine 支援向量機（Support Vector Machine，SVM）的核心精神為在二種類別 的資料之間找到一條很棒的分割線將二種類別的資料切割開來，並使資 料到分割線的邊界（Margins）最大化，讓不同類別的資料能完美區隔開 來。對於非線性的資料SVM 可以搭配適當的核函數（Kernel Function）， 將資料投影到更高維度的空間，在這個高維度空間內尋找合適的超平陎 （hyperplane）將資料區隔開來。SVM 除了單純處理分類的問題外，也 有延伸的SVR（Support Vector Regression）可以處理回歸的問題。 （3）Neural Network 類神經網路（Neural Network，NN）是一種模仿生物的神經網路（大腦） 的結構的學習系統。在生物學中神經系統由神經元（Neuron）構成，彼 此間透過突觸以電流傳遞訊號。而神經細胞會根據接收到的訊號量來決 定是否要傳遞訊號，當訊號量超過了某個閾值（Threshold）時，細胞便 會產生電流、通過突觸傳送給其他神經元。 一個類神經網路通常由許多層的神經元結構組成，而為了模擬神經細胞 的行為，會有一個激發函數（Activation Function）來決定神經元的行為， 對傳入的信號進行處理並決定是否要傳給下一層的神經元。每一份資料 透過輸進神經網路後，信號會一層層的傳遞下去直到最後一層輸出預測 結果，而根據預測的誤差來調整神經網路內部的權重，達到類似生物神 經學習的效果。 2. 簡易估測方法 （1）全部判斷為通關或滅團 根據每個頭目過往戰鬥的勝負來進行預測，來決定往後的戰鬥紀錄是通 關還是滅團。以副本中的第1 號頭目「聖光勇士」為例，超過8 成的隊 伍都能順利擊敗頭目，因此對於之後所有挑戰「聖光勇士」的團隊都全 部判定為通關。 （2）根據平均裝等決定勝負 以過往的戰鬥紀錄，根據副本中的每一個頭目，分別計算前來挑戰的團 隊的平均裝備等級，並以該平均裝等做為是否能通關的門檻。若是某團 隊裝等高於該門檻，則判定為通關，而低於門檻則判定為滅團。 四、實驗分析
