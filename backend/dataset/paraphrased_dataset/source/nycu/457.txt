本篇論文進行實驗的硬體配備為：Intel(R) Core(TM) i7-7700 CPU、GeForce GTX 1060 和GeForce GTX 1080 Ti 兩張GPU，使用的作業系統為Ubuntu 64-bit ( Linux )。使用到的程式語言包括python、Matlab 及C++ (配合OpenCV 2.4.11 輔助開發)，C++的開發環境主要是Microsoft Visual Studio 2015，主要用來處理 資料的前處理，而Matlab R2017b 主要用來作最終實驗結果準確率之統計與測 試，而訓練Mask R-CNN 的深度學習框架為Tensorflow 1.3、Keras 2.0.8。我們 的實驗中將模型學習率設為0.0002，而Batch size 為2。
