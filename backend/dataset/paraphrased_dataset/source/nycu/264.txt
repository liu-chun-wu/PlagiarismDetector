Bot，機器人( robot ) 的簡寫，他可以被預先寫好的指令控制。IRC bot 指的就 是利用IRC protocol 進行通訊與控制的受害主機，受害者會加入設定好的 IRC Server 與頻道，並依頻道中司令官的指令來做事。而 botnet，顧名思義就是由很多bots 所 組成，bots 之間可以互相通訊。如（圖 2） 所示為典型的IRC botnet 感染組成的週 期。Botnets 的 bot master 通常是透過感染主機上的漏洞來取得系統管理者所有權。 感染的途徑有很多種方式，像是惡意程式( malware )、蠕蟲( worms )，電子郵件病毒 ( e-mail viruses ) …等等。包括透過社交工程，電子郵件的附加檔案來說服受害者開 啟，以增加成功感染的機率。一旦被感染，受害者會從網路下載惡意程式來執行。 下載完成後，就會在受害者電腦上自動安裝，所以當電腦重新啟動時他就會隨著作 業系統自動執行了。 圖 2 典型的botnet 感染組成的週期
