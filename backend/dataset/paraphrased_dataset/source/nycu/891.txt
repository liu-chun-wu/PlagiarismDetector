Wi-Fi Mesh（或稱Mesh Wi-Fi）技術指的是：網路中每一台設備 都同時支援存取點以及路由器兩種模式，而在此網路中的每一台裝置 - 6 - 都不需要回到路由器做中介，就可以將每一台裝置互聯，因此每一個 存取點不但都能與主路由器連接，還能與其他存取點之間互相串連串 聯，藉此增大信號覆蓋的面積和強度。 由於現代家庭對Wi-Fi 的需求量增加，對於使用無線網路的效能 要求也提高，為了提供更好的QoS (Quality of Service)，就是Wi-Fi Mesh 所帶來的效益。Wi-Fi Mesh 能夠在多速率且多個AP（Access Point）的環境下提供最佳效能。當終端裝置在考慮多AP 的環境下， 此時該選擇哪一個AP 連線，就必須考慮到整體Throughput 與Delay 的影響，就是網路選擇(network selection)問題。在這個問題下，需要 訂定一個良好的網路架構，才能使得系統整體效能最大化。[5] 而在Wi-Fi Mesh 網路架構下，通過無線連接的方式，可以簡單佈 建且價格低廉，採用這種技術，可以很輕鬆的在城市，校園或體育場 建立無死角的網路。傳統的網絡必須依賴有線接點或無線熱點來連接 用戶。在WiFi Mesh 網絡架構下，網絡連接可以在廣大的區域，透過 幾十個甚至上百個無線網狀節點相互交談與分享資料。 WiFi Mesh網絡的最大優勢，相對於有線或無線固定式網絡而言， 是真正的實現了無線布局。傳統的無線固定式網路的AP 仍然需要透 過ADSL、專線(T1/E1)或是Cable 有線連接到Internet 來傳輸。而在 WiFi Mesh 網絡下，只需要一個有線節點，然後其他所有的AP 用最 接近他們的無線AP 共享，彼此進行無線共享其Internet 連接。 也就是說，Wi-Fi Mesh 為包含多個AP 的Wi-Fi 區域網路提供了 一種良好的解決方案，既具備易用、自我調整的優勢，又具備可交互 操作Wi-Fi 設備，提高了靈活性的優勢，藉此提供全面覆蓋室內和室 外空間的、智慧高效的Wi-Fi。 - 7 - Wi-Fi Mesh 安裝和使用都很簡便。網路的連線與加入新設備都僅 需最低限度的管理與設定。網路連線一旦建立完成，整個區域網路即 會進行自我監控，以確保整體效能。Wi-Fi Mesh 網路會利用Wi-Fi 聯 盟所制訂及提供的演算機制，將連線設備引導到能為其提供最佳效能 的AP 上。Wi-Fi Mesh 網路還可透過修改條件，改變網路結構，以提 供更優良的使用者體驗。 Wi-Fi Mesh 為住宅和辦公Wi-Fi 網路所提供的優勢如下：  靈活設計：允許以最佳方式放置多個AP，以提供更大的覆蓋 範圍。  易於設置：支援設備自動入網及配置。  網路智慧性：自我組織和自我優化的網路，可收集資訊，並 回應網路條件的變化，以最大限度提高性能。  有效的負載均衡：將設備漫遊到最佳的AP 上，以避免互相干 擾。  可擴展性：允許增加來自多個廠商的Wi-Fi Mesh AP。 總而言之，當人們在家中越來越依靠Wi-Fi 實現設備互連、提供 資料流傳送服務，就需要更加智慧的、覆蓋範圍更大的Wi-Fi 網路。 [6][7] - 8 - 圖1 Wi-Fi Easy Mesh 應用於一般家庭示意圖
