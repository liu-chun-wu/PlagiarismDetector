我們比較在三種不同數位設備輸入的條件(RWI,VRI 及VREI)及數位設備 輸入與傳統紙筆設備輸入下，完成每一項任務(Writing, Tracing, Drawing 及 Coloring)所需的平均時間。我們利用5(任務) X 3(RWI,VRI 及VREI) X 2(輸入設 備方式)的ANOVA 模型進行分析。分析指出不同的任務之間有顯著性的差異， F(4, 168) = 200.3, p < 0.001, η2 = 0.83，不同輸入設備之間有顯著性的差異， F(1, 168) = 67.67, p < 0.001, η2 = 0.62，及在不同輸入設備之下，進行不同任務 有顯著的差異，F(4, 168) = 68.63, p < 0.001, η2 = 0.62。我們利用Bonferroni 的 事後配對分析發現，用數位輸入設備進行實驗的任務平均花費時間較長，p < 0.001，如Fig. 22 所示，為每一個任務在不同輸入設備條件下所需時間的平均及 標準差。 Fig. 22 不同輸入條件下，進行不同任務所需的平均時間及平均時間標準差
